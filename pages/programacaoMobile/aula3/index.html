⚛️ O que é React?

Uma biblioteca JavaScript para criar interfaces de usuário (UI) de forma eficiente e declarativa.

Criada pelo Facebook em 2013.

Foco em componentes reutilizáveis.

Atualizações dinâmicas com Virtual DOM.

📖 Documentação Oficial

🛠️ Configurando o Ambiente

Passos básicos para começar com React:


# Instalar Node.js e npm
npx create-react-app meu-projeto --template typescript
cd meu-projeto
npm start

Use TypeScript para tipagem estática!

🧩 Componentes e JSX

Componentes são os blocos de construção do React.


// Exemplo de Componente com JSX
import React from 'react';

const MeuComponente: React.FC = () => {
  return 
Olá, React! 
;
};

export default MeuComponente;

JSX: HTML-like dentro do JavaScript.

📬 Props: Passando Dados

Props (properties) permitem enviar dados para componentes.


const Saudacao: React.FC<{ nome: string }> = ({ nome }) => {
  return 
Bem-vindo, {nome}!
;
};

// Uso

Props são imutáveis!
