Uso de StyleSheet

O StyleSheet é a forma mais básica e tradicional de definir estilos no React Native.

  • Permite criar um objeto de estilos em arquivo JS/TS.
  • Sintaxe simples e próxima do CSS.
  • Exemplo: 
    
                            import { StyleSheet, View, Text } from 'react-native';

                            const App = () => (
                              
                                Hello, StyleSheet!
                              
                            );
                            
                            const styles = StyleSheet.create({
                              container: { flex: 1, justifyContent: 'center', alignItems: 'center' },
                              text: { fontSize: 20, color: '#fff', fontWeight: 'bold' }

Por que usar StyleSheet?

  • Performance: Estilos são processados apenas uma vez.
  • Mais próximo do “tradicional” CSS, facilitando a adoção.
  • Melhor integração com animações simples.

Styled Components

Styled Components permite criar componentes de estilo dentro do próprio arquivo de JS/TS, mantendo estilo e lógica juntos.

  • Abordagem declarativa e modular
  • Reuso facilitado de componentes
  • Reduz a separação artificial entre JS e CSS

Documentação Oficial

Exemplo de Styled Components


import React from 'react';
import styled from 'styled-components/native';

const Container = styled.View`
  flex: 1;
  background-color: #222;
  align-items: center;
  justify-content: center;
`;

const Title = styled.Text`
  font-size: 24px;
  color: #fff;
`;

export default function App() {
  return (
    
      Olá Styled Components!
    
  );
}

Flexbox no React Native

  • Principais propriedades: flex, flexDirection, justifyContent, alignItems.
  • Por padrão, a direção é column (diferente do web que é row).
  • Permite criação de layouts responsivos e adaptativos.

Documentação Oficial Flexbox

Como o Flexbox organiza os elementos?

A organização vertical é padrão (coluna), mas você pode alterar para linha row facilmente.
Justifique e alinhe seus elementos para obter layouts sofisticados.

Flexbox no React Native

Criação de layouts flexíveis e adaptáveis

  • flexDirection: row, column
  • justifyContent: Distribuição de itens
  • alignItems: Alinhamento transversal

const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'row',
    justifyContent: 'space-between',
    alignItems: 'center',
  },
});

useWindowDimensions

O hook useWindowDimensions obtém as dimensões atuais da tela do dispositivo. Ótimo para criar layouts responsivos. 


import { useWindowDimensions } from 'react-native';

export default function MyComponent() {
  const { width, height } = useWindowDimensions();

  return (
    
      A largura da tela é: {width}
      A altura da tela é: {height}
    
  );
}

Prática: Construção de Layout Responsivo

  • Criar um contêiner principal com flex: 1.
  • Utilizar flexDirection para alternar entre "row" ou "column".
  • Aproveitar o useWindowDimensions para ajustes de elementos conforme o tamanho do dispositivo.
  • Exemplo de “cards” que se reordenam dinamicamente.
  • Aplicar margens e paddings dinâmicos para conteúdo, conforme a tela.

Passo a Passo Detalhado: Início da Prática

  • Configuração do Contêiner Principal:

    Crie um contêiner com flex: 1 para ocupar toda a tela.

    
const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#F0F8FF',
    justifyContent: 'center',
    alignItems: 'center',
  },
});
  • Uso do hook useWindowDimensions:

    Capture as dimensões da tela para tornar o layout responsivo.

    
import { useWindowDimensions } from 'react-native';

const { width, height } = useWindowDimensions();
  • Ajuste dinâmico do flexDirection:

    Defina a direção do layout com base na largura da tela.

    
const isLargeScreen = width > 600;
const flexDirection = isLargeScreen ? 'row' : 'column';
  • Cálculo de valores dinâmicos:

    Calcule padding e margin com base na largura da tela.

    
const dynamicPadding = width * 0.05;
const dynamicMargin = width * 0.02;
  • Criação dos "Cards":

    Estilize os cards com tamanhos que se ajustem dinamicamente.

    
    const cardSize = isLargeScreen ? 150 : 120;
