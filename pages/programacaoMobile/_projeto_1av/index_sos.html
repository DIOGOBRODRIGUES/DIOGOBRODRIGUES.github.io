Reeguer - Requisitos do Projeto e Ficha de Avaliação

Requisitos do Projeto

Descrição Geral

O aplicativo Reeguer conecta vítimas de desastres ambientais causados por chuvas (enchentes, deslizamentos) com pessoas ou organizações dispostas a ajudar. Inclui monitoramento básico, cadastro de necessidades, visualização de doações e lista de abrigos.

Requisitos Funcionais

  1. Tela de Login
    • Campos: Nome (texto simples) e senha (máscara para ocultar caracteres).
    • Botão "Entrar" que valida os dados (ex.: Nome "usuario" e senha "1234").
    • Mensagem de erro se os dados estiverem incorretos.
    • Redirecionar para a tela inicial após login.
  2. Tela Inicial (Dashboard)
    • Exibir notificação fixa (ex.: "Alerta: Risco de enchente em [cidade]").
    • Botões: "Pedir Ajuda", "Oferecer Ajuda" e "Abrigos".
    • Header com o nome "SOS Chuvas" e um ícone.
  3. Tela de Pedir Ajuda
    • Campos: Tipo de necessidade (dropdown: "Alimentos", "Roupas", "Abrigo") e descrição curta.
    • Botão "Enviar" exibe alerta confirmando o pedido.
    • Armazenar o pedido em uma lista local.
  4. Tela de Oferecer Ajuda
    • Lista (FlatList) com pedidos registrados (tipo e descrição).
    • Botão "Ajudar" em cada item exibe alerta com "Doação confirmada" ou "Voluntário confirmado".
  5. Tela de Abrigos
    • Lista (FlatList) com abrigos simulados (nome e endereço).
    • Itens estilizados com ícone ou cor diferente.

Requisitos Não Funcionais

  • Tecnologia: React Native com TypeScript ou JavaScript, usando React Navigation.
  • Estilização: Usar StyleSheets e componentes estilizáveis reutilizáveis.
  • Gerenciamento de Estado: Hooks (useState, useEffect); opcional: Context API.
  • Apresentação: Rodar no emulador ou dispositivo físico.

Escopo Mínimo

  • 4 telas: Login, Dashboard, Pedir Ajuda e Abrigos.
  • Validação básica nos formulários.
  • Navegação e estilização consistentes.

Opcionais (Extras)

  • Animações com Animated API (ex.: fade-in na notificação).
  • Logout na tela inicial.
  • AsyncStorage para salvar pedidos ou abrigos.
