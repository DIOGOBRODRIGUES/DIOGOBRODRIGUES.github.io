Atividade: Consumo da API Reqres.in e Implementação de um Front-end

Objetivo

Desenvolver uma aplicação front-end que consuma a API disponível em https://reqres.in/, utilizando todos os métodos HTTP (GET, POST, PUT, PATCH, DELETE).

Descrição da Atividade

  1. Familiarização com a API

    • Acesse a documentação da API Reqres.in e estude os endpoints disponíveis.
    • Entenda os dados retornados em cada requisição e os parâmetros necessários.

  2. Implementação do Front-end

    • Crie uma interface de usuário amigável que permita interagir com a API.
    • A interface deve contemplar as seguintes funcionalidades:
      • GET: Listar usuários e detalhes de um usuário específico.
      • POST: Criar um novo usuário.
      • PUT/PATCH: Atualizar informações de um usuário existente.
      • DELETE: Remover um usuário.

  3. Requisitos Técnicos

    • Utilize o frameworks React.
    • Implemente o tratamento de erros e feedback ao usuário (ex: mensagens de sucesso ou erro).

  4. Documentação e Boas Práticas

    • Comente o código explicando as principais funcionalidades.
    • Utilize padrões de projeto e organize o código de forma modular.

Observação: A resolução será discutida na próxima aula, onde poderemos esclarecer dúvidas e compartilhar experiências. Aproveite a oportunidade para explorar a integração com APIs e aprimorar suas habilidades em desenvolvimento front-end!
