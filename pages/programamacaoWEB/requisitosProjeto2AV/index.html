Instruções do Projeto de HTML, CSS e JavaScript

Objetivo

Desenvolver uma aplicação web responsiva utilizando HTML, CSS e JavaScript. O foco principal será nas funcionalidades em JavaScript, como a construção de um menu hambúrguer, manipulação da DOM e uso do Local Storage.

Requisitos do Projeto

  1. Estrutura HTML e Estilos CSS

    • Criar uma página web semântica e bem estruturada.
    • Aplicar estilos com CSS para melhorar a apresentação visual.
    • Garantir que o design seja responsivo em diferentes dispositivos.

  2. Funcionalidades em JavaScript

    • Menu Hambúrguer: Implementar um menu que seja visível e funcional em dispositivos móveis e desktops.
    • Manipulação da DOM: Utilizar JavaScript para alterar ou atualizar elementos na página em resposta a eventos do usuário.
    • Local Storage: Armazenar dados no navegador do usuário e recuperá-los posteriormente.

  3. Responsividade

    • Assegurar que a aplicação adapte seu layout de acordo com o tamanho da tela.

Esquema de Pontuação

  • Estrutura HTML correta: 0,4 ponto
  • Estilos CSS aplicados adequadamente: 0,4 ponto
  • Menu Hambúrguer funcional: 0,4 ponto
  • Manipulação da DOM implementada: 0,4 ponto
  • Uso do Local Storage: 0,4 ponto
  • Design responsivo: 0,4 ponto
  • Apresentação clara e objetiva: 0,6 ponto

Pontuação máxima: 2 pontos extras na prova.

Apresentação do Projeto

  • A apresentação ocorrerá em sala de aula, uma semana antes do dia da prova.
  • Cada grupo deverá demonstrar o funcionamento da aplicação e explicar o código desenvolvido.
  • Não é necessário preparar slides; o foco é no funcionamento e na explicação técnica.

Formação de Grupos

O projeto deve ser realizado em grupos de até 8 pessoas. Recomenda-se a colaboração e divisão de tarefas para um melhor aproveitamento.

Observações Importantes

  • O projeto é uma oportunidade para aplicar os conceitos aprendidos em aula.
  • A dedicação e o cumprimento dos requisitos podem contribuir significativamente para a nota final.
  • Este projeto valerá até 2 pontos extras na prova.
  • Todos os integrantes devem enviar o link do repositorio do projeto postado no github na tarefa do TEAM

Em caso de dúvidas, entre em contato com o professor.
