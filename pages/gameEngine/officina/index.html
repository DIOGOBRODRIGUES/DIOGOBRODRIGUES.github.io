Missão Pixel: Crie um Microjogo com Impacto Social

Prof. Dr. Diogo Francisco Borba Rodrigues

🎯 Tema da Oficina

“Missão Pixel: Crie um Microjogo com Impacto Social”

Desenvolver jogos com propósito social usando GDScript, criatividade e design interativo.

🌍 Narrativa de Envolvimento

“Você foi contratado para criar um jogo simples, mas com uma mensagem relevante sobre um tema social atual. Você tem apenas 3 horas para entregar!”

🛠 Requisitos Técnicos

  • Jogo funcional (máx. 2 minutos)
  • Início, objetivo e fim claros
  • Mensagem social integrada
  • Incluir pelo menos 2 elementos:
    • 🎮 Controle de personagem
    • 📦 Interação com objetos
    • ⭐ Pontuação ou vitória
    • 📣 Uso de signals
    • 📊 HUD simples

🧪 Temas Sugeridos

  • 🌱 Preservação ambiental
  • 💬 Combate ao cyberbullying
  • 🧠 Saúde mental e autocuidado
  • ♿ Acessibilidade e empatia
  • 🔐 Segurança na internet

✅ Critérios de Avaliação

  • 🎨 Criatividade e aderência ao tema
  • 🎮 Funcionalidade e jogabilidade
  • 📢 Clareza da mensagem
  • 🎧 Qualidade visual e sonora

🚀 Primeiros Passos: Instalação

  1. Acesse godotengine.org/download
  2. Baixe a versão Godot Engine - Standard (4.x)
  3. Versão para Windows: .exe ou .zip
  4. Versão para Mac: .dmg ou .zip
  5. Versão para Linux: .AppImage ou .tar.gz
  6. 💡 Godot não requer instalação! Execute diretamente o arquivo baixado

📂 Criando seu Primeiro Projeto

  1. Abra o Godot Engine
  2. Clique em "Novo Projeto"
  3. Dê um nome ao projeto (ex: "MissaoPlanetaLimpo")
  4. Escolha um diretório para salvar
  5. Selecione "Renderer: Forward+"
  6. Clique em "Criar e Editar"

💡 Salve o projeto em um local de fácil acesso!

🖥️ Interface do Godot

  • 📋 FileSystem: Arquivos e pastas do projeto
  • 🌳 Scene: Árvore de nós da cena atual
  • ⚙️ Inspector: Propriedades do nó selecionado
  • 👁️ Viewport: Visualização da cena (2D/3D)
  • 🎬 Scene Dock: Cenas disponíveis

💡 Pressione F1 para abrir a documentação!

🧩 Estrutura de Cenas

  • Godot usa um sistema de cenas e nós
  • Uma cena é composta por nós organizados em hierarquia
  • Para criar uma cena: Clique em "+" no editor de cenas
  • Nós principais para um microjogo 2D:
    • Node2D: Nó base para elementos 2D
    • Sprite2D: Imagens e gráficos
    • CharacterBody2D: Personagens controláveis
    • CollisionShape2D: Colisões
    • Area2D: Áreas de interação

📝 GDScript Básico


extends CharacterBody2D  # Herda de CharacterBody2D

# Variáveis
var speed = 300
var score = 0

# Função chamada a cada frame
func _process(delta):
    # Movimento do personagem
    var direction = Vector2.ZERO
    if Input.is_action_pressed("ui_right"):
        direction.x += 1
    if Input.is_action_pressed("ui_left"):
        direction.x -= 1
    
    velocity = direction.normalized() * speed
    move_and_slide()

📣 Signals (Sinais)

Signals permitem que nós se comuniquem sem referência direta.


# No nó que emite o sinal (ex: Area2D)
signal item_collected

func _on_body_entered(body):
    if body.name == "Player":
        emit_signal("item_collected")
        queue_free()  # Remove o nó

# No nó que recebe o sinal (ex: Player)
func _ready():
    $Area2D.connect("item_collected", _on_item_collected)

func _on_item_collected():
    score += 1
    print("Score: ", score)

🚢 Exportando seu Jogo

  1. Vá para "Projeto" > "Exportar..."
  2. Clique em "Adicionar..." e selecione a plataforma
  3. Configure as opções de exportação
  4. Clique em "Exportar Projeto..."
  5. Escolha o local e nome do arquivo

💡 Para Web/HTML5: Exporte para testar no navegador sem instalação!
