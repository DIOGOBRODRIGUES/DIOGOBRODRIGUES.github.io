🎯 Tema da Oficina

“Missão Pixel: Crie um Microjogo com Impacto Social”

Desenvolver jogos com propósito social usando GDScript, criatividade e design interativo.

🌍 Narrativa de Envolvimento

“Você foi contratado para criar um jogo simples, mas com uma mensagem relevante sobre um tema social atual. Você tem apenas 3 horas para entregar!”

🛠 Requisitos Técnicos

  • Jogo funcional (máx. 2 minutos)
  • Início, objetivo e fim claros
  • Mensagem social integrada
  • Incluir pelo menos 2 elementos:
    • 🎮 Controle de personagem
    • 📦 Interação com objetos
    • ⭐ Pontuação ou vitória
    • 📣 Uso de signals
    • 📊 HUD simples

🧪 Temas Sugeridos

  • 🌱 Preservação ambiental
  • 💬 Combate ao cyberbullying
  • 🧠 Saúde mental e autocuidado
  • ♿ Acessibilidade e empatia
  • 🔐 Segurança na internet

✅ Critérios de Avaliação

  • 🎨 Criatividade e aderência ao tema
  • 🎮 Funcionalidade e jogabilidade
  • 📢 Clareza da mensagem
  • 🎧 Qualidade visual e sonora

🚀 Primeiros Passos: Instalação

  1. Acesse godotengine.org/download
  2. Baixe a versão Godot Engine - Standard (4.x)
  3. Versão para Windows: .exe ou .zip
  4. Versão para Mac: .dmg ou .zip
  5. Versão para Linux: .AppImage ou .tar.gz
  6. 💡 Godot não requer instalação! Execute diretamente o arquivo baixado

📂 Criando seu Primeiro Projeto

  1. Abra o Godot Engine
  2. Clique em "Novo Projeto"
  3. Dê um nome ao projeto (ex: "MissaoPlanetaLimpo")
  4. Escolha um diretório para salvar
  5. Selecione "Renderer: Forward+"
  6. Clique em "Criar e Editar"

💡 Salve o projeto em um local de fácil acesso!

🖥️ Interface do Godot

  • 📋 FileSystem: Arquivos e pastas do projeto
  • 🌳 Scene: Árvore de nós da cena atual
  • ⚙️ Inspector: Propriedades do nó selecionado
  • 👁️ Viewport: Visualização da cena (2D/3D)
  • 🎬 Scene Dock: Cenas disponíveis

💡 Pressione F1 para abrir a documentação!

🧩 Estrutura de Cenas

  • Godot usa um sistema de cenas e nós
  • Uma cena é composta por nós organizados em hierarquia
  • Para criar uma cena: Clique em "+" no editor de cenas
  • Nós principais para um microjogo 2D:
    • Node2D: Nó base para elementos 2D
    • Sprite2D: Imagens e gráficos
    • CharacterBody2D: Personagens controláveis
    • CollisionShape2D: Colisões
    • Area2D: Áreas de interação

📝 GDScript Básico


extends CharacterBody2D  # Herda de CharacterBody2D

# Variáveis
var speed = 300
var score = 0

# Função chamada a cada frame
func _process(delta):
    # Movimento do personagem
    var direction = Vector2.ZERO
    if Input.is_action_pressed("ui_right"):
        direction.x += 1
    if Input.is_action_pressed("ui_left"):
        direction.x -= 1
    
    velocity = direction.normalized() * speed
    move_and_slide()

📣 Signals (Sinais)

Signals permitem que nós se comuniquem sem referência direta.


# No nó que emite o sinal (ex: Area2D)
signal item_collected

func _on_body_entered(body):
    if body.name == "Player":
        emit_signal("item_collected")
        queue_free()  # Remove o nó

# No nó que recebe o sinal (ex: Player)
func _ready():
    $Area2D.connect("item_collected", _on_item_collected)

func _on_item_collected():
    score += 1
    print("Score: ", score)

👤 Criando um Personagem Controlável

Vamos criar um personagem que o jogador poderá controlar

Use as setas ↓ para ver os passos detalhados

1. Criando a Estrutura Básica

  1. Abra o Godot e selecione seu projeto
  2. Pressione Ctrl+N (ou Cmd+N no Mac) para criar nova cena
  3. Selecione "CharacterBody2D" como nó raiz
  4. No painel inspetor, renomeie para "Player"

2. Adicionando Componentes Visuais

  1. Clique com botão direito no nó "Player"
  2. Selecione "Adicionar Nó Filho"
  3. Escolha "Sprite2D"
  4. No Inspector, clique em "Texture" → "Carregar"
  5. Selecione uma imagem para seu personagem
  6. Se não tiver, use os recursos da seção Assets

3. Adicionando Colisões

  1. Clique com botão direito no nó "Player" novamente
  2. Adicione um nó filho "CollisionShape2D"
  3. No Inspector, clique em "Shape" → "New RectangleShape2D"
  4. Ajuste o tamanho da colisão para corresponder ao sprite
  5. Use as alças de redimensionamento no editor
  6. Salve a cena: Ctrl+S (ou Cmd+S)
  7. Nome sugerido: "Player.tscn"

🎮 Programando o Movimento

Agora vamos fazer o personagem se movimentar com as teclas direcionais

Use as setas ↓ para ver os passos detalhados

1. Criando o Script

  1. Selecione o nó raiz "Player" na árvore de cenas
  2. No painel Inspector, localize o botão com ícone de script
  3. Clique em "Attach Script"
  4. Mantenha as opções padrão:
    • Linguagem: GDScript
    • Modelo: Node
    • Nome: Player.gd
  5. Clique em "Create"

2. Código para Movimentação (Parte 1)


extends CharacterBody2D

# Propriedades do personagem
var speed = 300  # Velocidade de movimento em pixels/segundo

# Esta função é chamada a cada quadro físico do jogo
func _physics_process(delta):
    # Criamos um vetor para armazenar a direção
    var direction = Vector2.ZERO
    
    # Detectamos as teclas pressionadas
    if Input.is_action_pressed("ui_right"):
        direction.x += 1
    if Input.is_action_pressed("ui_left"):
        direction.x -= 1
    if Input.is_action_pressed("ui_down"):
        direction.y += 1
    if Input.is_action_pressed("ui_up"):
        direction.y -= 1

As teclas "ui_right", "ui_left", "ui_down", "ui_up" representam as setas do teclado

2. Código para Movimentação (Parte 2)


    # Normaliza o vetor para movimento consistente em diagonais
    if direction.length() > 0:
        direction = direction.normalized()
    
    # Define a velocidade do personagem
    velocity = direction * speed
    
    # Move o personagem e lida com colisões automaticamente
    move_and_slide()
    
    # Você pode adicionar animações aqui (opcional)
    # if direction.x != 0:
    #     $Sprite2D.flip_h = direction.x < 0

💡 Dica: Teste seu personagem pressionando F5 para executar o jogo

3. Explicação do Código

  • extends CharacterBody2D: Herda funcionalidades do nó CharacterBody2D
  • _physics_process(delta): Função chamada a cada quadro de física
  • direction: Vetor que define a direção do movimento
  • direction.normalized(): Mantém a mesma velocidade em diagonais
  • velocity: Propriedade herdada que define a velocidade
  • move_and_slide(): Move o personagem e lida com colisões

Com isso, seu personagem já deve ser capaz de se movimentar com as setas do teclado!

🌍 Criando a Cena Principal

Vamos criar o ambiente onde o jogo acontecerá

🍎 Criando Objetos Coletáveis

Vamos criar itens que o jogador poderá coletar

Use as setas ↓ para ver os passos detalhados

1. Criando a Cena do Objeto

  1. Pressione Ctrl+N para criar nova cena
  2. Selecione "Area2D" como nó raiz
  3. Renomeie para "Coletavel"
  4. Adicione um nó filho "Sprite2D"
  5. No Inspector, defina uma textura para o sprite
  6. Escolha uma imagem relacionada ao tema social

Exemplos por tema:

  • Ambiente: garrafas, latas, folhas
  • Saúde: frutas, remédios, kit médico
  • Educação: livros, lápis, computador
  • Acessibilidade: símbolos de inclusão

2. Adicionando Colisão ao Objeto

  1. Adicione um nó filho "CollisionShape2D" ao "Coletavel"
  2. No Inspector, clique em "Shape" → "New CircleShape2D"
  3. Ajuste o tamanho para corresponder ao sprite
  4. Salve a cena: Ctrl+S
  5. Nome sugerido: "Coletavel.tscn"

💡 CircleShape2D é geralmente melhor para coletáveis porque funciona bem independente da rotação

3. Criando o Script do Coletável


extends Area2D

signal coletado  # Define um sinal personalizado

func _ready():
    # Conecta o sinal interno de colisão à nossa função
    body_entered.connect(_on_body_entered)

func _on_body_entered(body):
    # Verifica se o que colidiu é o jogador
    if body.name == "Player" or body.get_parent().name == "Player":
        # Emite nosso sinal personalizado
        emit_signal("coletado")
        
        # Remove o objeto da cena
        queue_free()

Este código detecta quando o jogador toca o objeto e o remove da cena

🎯 Criando o Sistema de Pontuação

Vamos criar uma interface para mostrar os pontos do jogador

Use as setas ↓ para ver os passos detalhados

1. Adicionando a Interface (HUD)

  1. Na cena "Main", adicione um nó "CanvasLayer"
    • CanvasLayer permite que elementos fiquem fixos na tela
  2. Renomeie para "HUD"
  3. Adicione um nó filho "Label"
  4. No Inspector, configure:
    • Text: "Pontuação: 0"
    • Position: (20, 20) aproximadamente
    • Theme Override → Font Size: 22

2. Script para Controlar a Pontuação

  1. Selecione o nó raiz "Main"
  2. Adicione um script (attach script)
  3. Mantenha as opções padrão e clique em "Create"

extends Node2D

# Variáveis
var pontuacao = 0
@onready var pontuacao_label = $HUD/Label

# Chamado quando a cena é carregada
func _ready():
    # Atualiza o texto inicial
    atualizar_pontuacao()
    
    # Vai ser chamado quando adicionarmos coletáveis
    # Por enquanto está vazio

3. Função para Atualizar Pontuação


# Adicione esta função ao script da cena Main
func atualizar_pontuacao():
    pontuacao_label.text = "Pontuação: " + str(pontuacao)

# Esta função será chamada quando um item for coletado
func _on_coletavel_coletado():
    # Aumenta a pontuação
    pontuacao += 1
    
    # Atualiza o texto na interface
    atualizar_pontuacao()
    
    # Verificar condição de vitória (opcional)
    if pontuacao >= 10:  # Ajuste conforme necessário
        mostrar_mensagem_vitoria()

💡 Por enquanto, nosso sistema está pronto mas não conectado aos coletáveis

🔗 Conectando os Elementos

Vamos juntar todos os componentes e fazer o jogo funcionar

Use as setas ↓ para ver os passos detalhados

1. Adicionando Coletáveis à Cena Principal

  1. Na cena "Main", use o botão "Instanciar Cena" (ícone de corrente)
  2. Selecione "Coletavel.tscn"
  3. Clique várias vezes no editor para adicionar múltiplos objetos
  4. Posicione os objetos em diferentes locais da cena
  5. Selecione cada coletável e, no Inspector:
    • Vá para a guia "Node"
    • Na seção "Groups", digite "coletaveis"
    • Clique em "Add"

Isso adiciona cada objeto ao grupo "coletaveis" para fácil referência

2. Conectando os Sinais

Vamos modificar o script da cena Main para conectar os coletáveis:


# No script Main.gd, modifique a função _ready:

func _ready():
    # Atualiza o texto inicial
    atualizar_pontuacao()
    
    # Conecta o sinal de cada coletável
    for coletavel in get_tree().get_nodes_in_group("coletaveis"):
        coletavel.connect("coletado", _on_coletavel_coletado)

Este código encontra todos os objetos do grupo "coletaveis" e conecta seus sinais à nossa função de pontuação

3. Testando o Jogo

  1. Salve todas as cenas: Ctrl+Shift+S
  2. Pressione F5 para executar o jogo
  3. Use as setas direcionais para mover o personagem
  4. Colete os objetos e observe a pontuação aumentar

Parabéns! Você criou um jogo básico com coletáveis!

💭 Integrando a Mensagem Social

Vamos adicionar sentido ao jogo conectando-o ao tema social escolhido

Use as setas ↓ para ver as opções e exemplos

1. Métodos para Integrar a Mensagem

Três abordagens principais para transmitir sua mensagem social:

  • Narrativa - Através da história do jogo
  • Mecânicas - Através das regras e interações
  • Visual - Através de elementos gráficos e cenário

Detalhes nas próximas telas ↓

1.1. Abordagem Narrativa

Integre sua mensagem através da história:

  • Tela inicial com contexto do problema social
  • Diálogos durante o jogo em momentos-chave
  • Textos explicativos conectando ações à realidade

1.2. Abordagem de Mecânicas

A mensagem emerge da jogabilidade:

  • Coletáveis representam ações positivas
  • Obstáculos simbolizam problemas reais
  • Progresso no jogo reflete mudança social

1.3. Abordagem Visual

Comunique a mensagem através do visual:

  • Ambiente evolui com o progresso (ex: de poluído para limpo)
  • Personagens e objetos simbolizam elementos do tema
  • Cores e símbolos reforçam a mensagem

2. Exemplo: Tela com Mensagem Educativa

  1. Na cena "Main", adicione um nó filho "CanvasLayer"
  2. Renomeie para "MensagemFinal"
  3. Adicione um filho "ColorRect" para o fundo
  4. Configure no Inspector:
    • Anchor/Margins: Preencha toda a tela
    • Color: Escolha uma cor com transparência
  5. Adicione um nó filho "Label" ao ColorRect
  6. Configure texto informativo e educativo sobre o tema
  7. Adicione um botão "Jogar Novamente" se desejar
  8. Defina MensagemFinal.visible = false inicialmente

3. Mostrando a Mensagem no Final

Adicione esta função ao script da cena Main:


# Função para mostrar mensagem educativa no final
func mostrar_mensagem_vitoria():
    # Pausa o jogo
    get_tree().paused = true
    
    # Torna a mensagem visível
    $MensagemFinal.visible = true
    
    # Define o texto educativo baseado no tema
    $MensagemFinal/ColorRect/Label.text = """
    Parabéns! Você coletou {0} itens recicláveis.
    
    Sabia que? No Brasil, apenas 3% de todo o lixo produzido
    é reciclado, enquanto que em países desenvolvidos
    esse número pode chegar a 60%.
    
    Pequenas ações como separar o lixo reciclável
    fazem uma grande diferença para o planeta!
    
    VOCÊ pode ser parte da solução!
    """.format([pontuacao])

Personalize esta mensagem conforme o tema social escolhido
