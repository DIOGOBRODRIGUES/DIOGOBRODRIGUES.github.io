📁 Estrutura de Pastas

Comparação entre Expo e React Native CLI

Expo 📦

Estrutura simplificada, ferramentas prontas para uso

React Native CLI 🛠️

Maior controle, configuração manual

📁 Estrutura Básica Expo


projeto-expo/
├── assets/              # Imagens, fontes e outros recursos
├── app.json             # Configuração do Expo
├── App.js               # Ponto de entrada da aplicação
├── components/          # Componentes reutilizáveis
├── screens/             # Telas da aplicação
├── navigation/          # Navegação entre telas
├── services/            # Serviços e APIs
├── package.json         # Dependências do projeto
├── node_modules/        # Módulos instalados
└── .gitignore           # Arquivos ignorados pelo git

O Expo facilita a configuração inicial e oferece ferramentas como Expo Go para testes rápidos

📁 Estrutura React Native CLI


projeto-rn-cli/
├── android/             # Código nativo Android
├── ios/                 # Código nativo iOS
├── src/                 # Código fonte principal
│   ├── assets/          # Recursos estáticos
│   ├── components/      # Componentes reutilizáveis
│   ├── screens/         # Telas da aplicação
│   ├── navigation/      # Configuração de rotas
│   ├── services/        # Serviços e APIs
│   ├── utils/           # Funções utilitárias
│   └── App.js           # Componente principal
├── index.js             # Ponto de entrada
├── package.json         # Dependências do projeto
├── metro.config.js      # Configuração do Metro Bundler
├── babel.config.js      # Configuração do Babel
└── node_modules/        # Módulos instalados

O CLI oferece acesso direto ao código nativo, ideal para recursos específicos de plataforma

🎨 Estilização com StyleSheet

O React Native utiliza JavaScript para definir estilos



import { View, Text, StyleSheet } from 'react-native';

// Componente com estilos
const Card = () => (
  <View style={styles.card}>
    <Text style={styles.title}>Meu Cartão</Text>
    <Text style={styles.description}>
      Este é um exemplo de cartão estilizado com StyleSheet
    </Text>
  </View>
);

// Definição de estilos
const styles = StyleSheet.create({
  card: {
    backgroundColor: '#ffffff',
    borderRadius: 8,
    padding: 16,
    margin: 8,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
    shadowRadius: 4,
    elevation: 3, // Para Android
  },
  title: {
    fontSize: 18,
    fontWeight: 'bold',
    marginBottom: 8,
  },
  description: {
    fontSize: 14,
    color: '#666666',
  },
});

export default Card;

💪 Vantagens do StyleSheet

  • Otimização de performance (cria referências de ID)
  • Autocomplete em editores modernos
  • Separação entre lógica e estilo

⚠️ As propriedades CSS tradicionais são adaptadas para camelCase!


// Web CSS:    background-color, font-size
// React Native: backgroundColor, fontSize

🎯 Aplicação de Estilos

Múltiplas formas de aplicar estilos em componentes:


// 1. Estilo único
<View style={styles.container}>
  <Text>Texto simples</Text>
</View>

// 2. Múltiplos estilos (array)
<View style={[styles.container, styles.padded]}>
  <Text>Combinação de estilos</Text>
</View>

// 3. Estilos condicionais
<View style={[
  styles.button, 
  isActive ? styles.activeButton : styles.inactiveButton
]}>
  <Text>Botão com estado</Text>
</View>

// 4. Estilos inline (evitar para código mais limpo)
<View style={{ 
  padding: 20, 
  backgroundColor: isDark ? '#000' : '#fff'
}}>
  <Text>Estilos inline</Text>
</View>

🔄 Estilização Dinâmica


import { useState } from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

const DynamicStyleExample = () => {
  const [isActive, setIsActive] = useState(false);
  
  const toggleActive = () => {
    setIsActive(!isActive);

  return (
    <View style={styles.container}>

        style={[
          styles.button,

          isActive ? styles.activeText : styles.inactiveText

💅 Styled Components

Uma abordagem alternativa para estilização usando CSS-in-JS


// Instalação:
// npm install styled-components
// ou
// yarn add styled-components
import { SafeAreaView} from 'react-native';
import styled from 'styled-components/native';

// Definindo componentes estilizados
const Container = styled.View`
  background-color: #ffffff;
  padding: 16px;
  border-radius: 8px;
  margin: 8px;
  shadow-opacity: 0.1;
  shadow-radius: 4px;
  shadow-color: #000;
  shadow-offset: 0px 2px;
  elevation: 3;
`;

const Title = styled.Text`
  font-size: 18px;
  font-weight: bold;
  margin-bottom: 8px;
  color: ${props => props.primary ? '#1e88e5' : '#333'};
`;

const Description = styled.Text`
  font-size: 14px;
  color: #666666;
`;
// Componente com Styled Components
const StyledCard = ({ title, description, isPrimary }) => (
  <Container>
    <Title primary={isPrimary}>{title}</Title>
    <Description>{description}</Description>
  </Container>
);

// Uso do componente
const App = () => (
  <SafeAreaView>
    <StyledCard 
      title="Meu Cartão Estilizado" 
      description="Usando Styled Components para estilizar"
      isPrimary={true}
    />
  </SafeAreaView>
);

📊 StyleSheet vs Styled Components

StyleSheet 📋

  • Nativo do React Native
  • Performance otimizada
  • Tipagem forte com TypeScript
  • Validações em tempo de compilação
  • Sem dependências externas

📊 StyleSheet vs Styled Components

Styled Components 💅

  • Sintaxe CSS tradicional
  • Props dinâmicas para estilos
  • Melhor reutilização de componentes
  • Temas globais
  • Bibliotecas de UI familiares para desenvolvedores web

📏 Flexbox em React Native

Flexbox é o principal sistema de layout em React Native

Principais propriedades:

  • flex
  • flexDirection
  • justifyContent
  • alignItems
  • flexWrap

Propriedades complementares:

  • alignSelf
  • flexBasis
  • flexGrow
  • flexShrink
  • position

📊 Propriedades Principais


// flexDirection: define o eixo principal
const styles = StyleSheet.create({
  row: {
    flexDirection: 'row', // horizontal (padrão no web é 'column')
  },
  column: {
    flexDirection: 'column', // vertical (padrão no React Native)
  },
  rowReverse: {
    flexDirection: 'row-reverse',
  },
  columnReverse: {
    flexDirection: 'column-reverse',
  },
  
  // justifyContent: alinhamento no eixo principal
  justifyStart: {
    justifyContent: 'flex-start', // padrão
  },
  justifyCenter: {
    justifyContent: 'center',
  },
  justifyEnd: {
    justifyContent: 'flex-end',
  },
  justifySpaceBetween: {
    justifyContent: 'space-between',
  },
  justifySpaceAround: {
    justifyContent: 'space-around',
  },
  
  // alignItems: alinhamento no eixo secundário
  alignStart: {
    alignItems: 'flex-start',
  },
  alignCenter: {
    alignItems: 'center',

  },

  },
});

📐 useWindowDimensions

Hook para obter dimensões da janela e criar layouts responsivos



import { View, Text, useWindowDimensions, StyleSheet } from 'react-native';

// O hook retorna as dimensões atuais da janela
// e atualiza automaticamente quando há mudanças de orientação
// ou redimensionamento (em alguns dispositivos)

const ResponsiveLayout = () => {
  // Obtendo as dimensões da tela
  const { width, height } = useWindowDimensions();
  
  // Definindo layout baseado na orientação
  const isLandscape = width > height;
  
  return (
    <View style={styles.container}>
      <Text style={styles.text}>
        Largura da tela: {width}px
      </Text>
      <Text style={styles.text}>
        Altura da tela: {height}px
      </Text>
      <Text style={styles.text}>
        Orientação: {isLandscape ? 'Paisagem' : 'Retrato'}
      </Text>
      
      <View style={[
        styles.box, 
        { 
          // Largura responsiva: menor em paisagem, maior em retrato
          width: isLandscape ? width * 0.3 : width * 0.8,
          backgroundColor: isLandscape ? '#3498db' : '#e74c3c'
        }
      ]}>
        <Text style={styles.boxText}>
          Este elemento se adapta à orientação
        </Text>
      </View>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
  text: {
    fontSize: 16,
    marginBottom: 10,
  },
  box: {
    marginTop: 20,
    padding: 20,
    borderRadius: 8,
    justifyContent: 'center',
    alignItems: 'center',
  },
  boxText: {
    color: 'white',
    fontWeight: 'bold',

📝 Formulários em React Native

Implementando formulários eficientes e validados


import { useState } from 'react';
import { 
  View, 
  Text, 
  TextInput, 
  TouchableOpacity, 
  StyleSheet,
  Alert,
  KeyboardAvoidingView,
  Platform
} from 'react-native';

const SimpleForm = () => {
  // Estado para os campos do formulário
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    password: '',
  });
  
  // Estado para os erros
  const [errors, setErrors] = useState({
    name: '',
    email: '',
    password: '',
  });
  
  // Atualização do estado do formulário
  const handleChange = (field, value) => {
    setFormData({
      ...formData,
      [field]: value
    });
    
    // Limpa o erro quando o usuário começa a digitar
    if (errors[field]) {
      setErrors({
        ...errors,
        [field]: ''
      });
    }
  };
  
  // Validação de formulário
  const validateForm = () => {
    let isValid = true;
    const newErrors = { name: '', email: '', password: '' };
    
    // Validação do nome
    if (!formData.name.trim()) {
      newErrors.name = 'Nome é obrigatório';
      isValid = false;
    }
    
    // Validação de email
    if (!formData.email) {
      newErrors.email = 'Email é obrigatório';
      isValid = false;
    } else if (!/\S+@\S+\.\S+/.test(formData.email)) {
      newErrors.email = 'Email inválido';
      isValid = false;
    }
    
    // Validação de senha
    if (!formData.password) {
      newErrors.password = 'Senha é obrigatória';
      isValid = false;
    } else if (formData.password.length < 6) {
      newErrors.password = 'A senha deve ter pelo menos 6 caracteres';
      isValid = false;
    }
    
    setErrors(newErrors);
    return isValid;
  };

📱 Interface do Formulário


  // Submissão do formulário
  const handleSubmit = () => {
    if (validateForm()) {
      Alert.alert(
        'Sucesso', 
        'Formulário enviado com sucesso!',
        [{ text: 'OK' }]
      );
      // Aqui você pode enviar os dados para uma API
      console.log('Dados do formulário:', formData);
    } else {
      Alert.alert(
        'Erro', 
        'Por favor, corrija os erros no formulário.',
        [{ text: 'OK' }]
      );
    }
  };
  
  return (
    <KeyboardAvoidingView 
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
      style={styles.container}
    >
      <View style={styles.form}>
        <Text style={styles.header}>Formulário de Cadastro</Text>
        
        <Text style={styles.label}>Nome:</Text>
        <TextInput
          style={[styles.input, errors.name ? styles.inputError : null]}
          placeholder="Digite seu nome"
          value={formData.name}
          onChangeText={(text) => handleChange('name', text)}
        />
        {errors.name ? <Text style={styles.errorText}>{errors.name}</Text> : null}
        
        <Text style={styles.label}>Email:</Text>
        <TextInput
          style={[styles.input, errors.email ? styles.inputError : null]}
          placeholder="Digite seu email"
          value={formData.email}
          onChangeText={(text) => handleChange('email', text)}
          keyboardType="email-address"
          autoCapitalize="none"
        />
        {errors.email ? <Text style={styles.errorText}>{errors.email}</Text> : null}
        
        <Text style={styles.label}>Senha:</Text>
        <TextInput
          style={[styles.input, errors.password ? styles.inputError : null]}
          placeholder="Digite sua senha"
          value={formData.password}
          onChangeText={(text) => handleChange('password', text)}
          secureTextEntry
        />
        {errors.password ? <Text style={styles.errorText}>{errors.password}</Text> : null}
        
        <TouchableOpacity style={styles.button} onPress={handleSubmit}>
          <Text style={styles.buttonText}>Cadastrar</Text>
        </TouchableOpacity>
      </View>
    </KeyboardAvoidingView>
  );
};
