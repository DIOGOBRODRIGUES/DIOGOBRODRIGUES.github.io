🔄 Loops Aninhados

Quando uma estrutura de repetição é colocada dentro de outra

Principais características:

  • Loop externo controla o número de iterações do loop interno
  • Para cada iteração do loop externo, o loop interno executa totalmente
  • Excelente para trabalhar com estruturas bidimensionais

⬇️ Deslize para baixo para ver exemplos

🔄 Estrutura Básica de Loops Aninhados

algoritmo "LoopsAninhados"
var
   i, j: inteiro
inicio
   para i de 1 ate 3 faca
      para j de 1 ate 2 faca
         escreval("i =", i, " e j =", j)
      fimpara
      escreval("") // Quebra de linha
   fimpara
fimalgoritmo

Saída:

i = 1 e j = 1
i = 1 e j = 2
i = 2 e j = 1
i = 2 e j = 2
i = 3 e j = 1
i = 3 e j = 2

📊 Vetores em Visualg

Vetores são estruturas que armazenam sequências de valores do mesmo tipo

  • Também conhecidos como arrays unidimensionais
  • Cada elemento é acessado por um índice numérico
  • No Visualg, os índices começam em 1
  • Facilita o armazenamento e manipulação de conjuntos de dados

⬇️ Deslize para baixo para exemplos práticos

📊 Declaração e Uso Básico de Vetores

algoritmo "VetoresBasico"
var
   notas: vetor[1..5] de real
   i: inteiro
   soma, media: real
inicio
   // Leitura das notas
   para i de 1 ate 5 faca
      escreva("Digite a ", i, "ª nota: ")
      leia(notas[i])
      soma <- soma + notas[i]
   fimpara
   
   // Calculando a média
   media <- soma / 5
   escreval("A média das notas é: ", media)
   
   // Exibindo todas as notas
   escreval("Notas digitadas:")
   para i de 1 ate 5 faca
      escreval("Nota ", i, ": ", notas[i])
   fimpara
fimalgoritmo

🎮 Exemplo: Jogo da Adivinhação

algoritmo "JogoAdivinhacao"
var
   segredos: vetor[1..5] de inteiro
   palpite, i, acertos: inteiro
inicio
   // Inicializando os números secretos
   segredos[1] <- 42
   segredos[2] <- 7
   segredos[3] <- 13
   segredos[4] <- 21
   segredos[5] <- 29
   
   acertos <- 0
   escreval("🎮 JOGO DA ADIVINHAÇÃO 🎮")
   escreval("Tente adivinhar um dos 5 números secretos")
   escreva("Digite seu palpite: ")
   leia(palpite)
   
   para i de 1 ate 5 faca
      se (palpite = segredos[i]) entao
         acertos <- acertos + 1
      fimse
   fimpara
   
   se (acertos > 0) entao
      escreval("🎉 PARABÉNS! Você acertou!")
   senao
      escreval("❌ Que pena! Tente novamente.")
   fimse
fimalgoritmo

🔍 Busca em Vetores

algoritmo "BuscaEmVetor"
var
   nomes: vetor[1..5] de caractere
   procurado: caractere
   i, posicao: inteiro
   encontrado: logico
inicio
   // Preenchendo o vetor
   nomes[1] <- "Ana"
   nomes[2] <- "Bruno"
   nomes[3] <- "Carla"
   nomes[4] <- "Daniel"
   nomes[5] <- "Eduarda"
   
   // Busca
   escreva("Digite um nome para buscar: ")
   leia(procurado)
   
   encontrado <- falso
   posicao <- 0
   
   para i de 1 ate 5 faca
      se (nomes[i] = procurado) entao
         encontrado <- verdadeiro
         posicao <- i
         interrompa
      fimse
   fimpara
   
   se (encontrado) entao
      escreval("✅ Nome encontrado na posição ", posicao)
   senao
      escreval("❌ Nome não encontrado!")
   fimse
fimalgoritmo

🧩 Matrizes em Visualg

Matrizes são estruturas bidimensionais que armazenam dados em linhas e colunas

  • Arrays bidimensionais - podem ser visualizados como tabelas
  • Acessadas usando dois índices: linha e coluna
  • Úteis para representar dados tabulares, grades e sistemas de coordenadas
  • Indicadas para jogos, cálculos matemáticos e organização de informações

⬇️ Deslize para baixo para ver exemplos

🧩 Declaração e Uso Básico de Matrizes

algoritmo "MatrizBasica"
var
   matriz: vetor[1..3, 1..3] de inteiro
   i, j: inteiro
inicio
   // Preenchendo a matriz
   para i de 1 ate 3 faca
      para j de 1 ate 3 faca
         escreva("Digite o valor da posição [", i, ",", j, "]: ")
         leia(matriz[i, j])
      fimpara
   fimpara
   
   // Exibindo a matriz formatada
   escreval("Matriz digitada:")
   para i de 1 ate 3 faca
      para j de 1 ate 3 faca
         escreva(matriz[i, j]:4) // Formata com 4 espaços
      fimpara
      escreval() // Quebra de linha ao final de cada linha
   fimpara
fimalgoritmo

🎲 Exemplo: Jogo da Velha

algoritmo "JogoDaVelha"
var
   tabuleiro: vetor[1..3, 1..3] de caractere
   i, j, jogadas: inteiro
   jogador: caractere
   linha, coluna: inteiro
   ganhou: logico
inicio
   // Inicializando o tabuleiro vazio
   para i de 1 ate 3 faca
      para j de 1 ate 3 faca
         tabuleiro[i, j] <- " "
      fimpara
   fimpara
   
   jogador <- "X" // Primeiro jogador é o X
   jogadas <- 0
   ganhou <- falso
   
   enquanto (jogadas < 9) e (nao ganhou) faca
      // Exibindo o tabuleiro
      limpatela
      escreval("  JOGO DA VELHA")
      escreval("----------------")
      para i de 1 ate 3 faca
         escreva(" |")
         para j de 1 ate 3 faca
            escreva(" ", tabuleiro[i, j], " |")
         fimpara
         escreval("")
         escreval("----------------")
      fimpara
      
      // Lendo a jogada
      escreval("Jogador ", jogador, ", sua vez!")
      escreva("Linha (1-3): ")
      leia(linha)
      escreva("Coluna (1-3): ")
      leia(coluna)
      
      // Verificando se a jogada é válida
      se (tabuleiro[linha, coluna] = " ") entao
         tabuleiro[linha, coluna] <- jogador
         jogadas <- jogadas + 1
         
         // Verificar se ganhou (simplificado)
         // ... código de verificação da vitória...
         
         // Alternar jogador
         se (jogador = "X") entao
            jogador <- "O"
         senao
            jogador <- "X"
         fimse
      senao
         escreval("Posição ocupada! Tente novamente.")
         Util.Aguarde(2000)
      fimse

📊 Exemplo: Soma de Matrizes

algoritmo "SomaMatrizes"
var
   A, B, C: vetor[1..2, 1..3] de inteiro
   i, j: inteiro
inicio
   // Lendo a primeira matriz
   escreval("🔢 Digite os valores da matriz A:")
   para i de 1 ate 2 faca
      para j de 1 ate 3 faca
         escreva("A[", i, ",", j, "]: ")
         leia(A[i, j])
      fimpara
   fimpara
   
   // Lendo a segunda matriz
   escreval("🔢 Digite os valores da matriz B:")
   para i de 1 ate 2 faca
      para j de 1 ate 3 faca
         escreva("B[", i, ",", j, "]: ")
         leia(B[i, j])
      fimpara
   fimpara
   
   // Somando as matrizes
   para i de 1 ate 2 faca
      para j de 1 ate 3 faca
         C[i, j] <- A[i, j] + B[i, j]
      fimpara
   fimpara
   
   // Exibindo a matriz resultante
   escreval("📊 Matriz Resultante (A + B):")
   para i de 1 ate 2 faca
      escreva("| ")
      para j de 1 ate 3 faca
         escreva(C[i, j]:4)
      fimpara
      escreval(" |")
   fimpara
fimalgoritmo

🌟 Integrando Loops, Vetores e Matrizes

algoritmo "SistemaNotas"
var
   turmas: vetor[1..3, 1..5] de real  // 3 turmas, 5 alunos cada
   medias: vetor[1..3] de real        // média de cada turma
   i, j: inteiro
   soma, mediaGeral: real
inicio
   // Entrada de dados das notas
   para i de 1 ate 3 faca
      escreval("📚 Turma ", i, ":")
      para j de 1 ate 5 faca
         escreva("   Nota do aluno ", j, ": ")
         leia(turmas[i, j])
      fimpara
   fimpara
   
   // Cálculo das médias por turma
   para i de 1 ate 3 faca
      soma <- 0
      para j de 1 ate 5 faca
         soma <- soma + turmas[i, j]
      fimpara
      medias[i] <- soma / 5
   fimpara
   
   // Cálculo da média geral
   soma <- 0
   para i de 1 ate 3 faca
      soma <- soma + medias[i]
   fimpara
   mediaGeral <- soma / 3
   
   // Exibição dos resultados
   escreval("")
   escreval("📊 RESULTADOS:")
   para i de 1 ate 3 faca
      escreval("Média da Turma ", i, ": ", medias[i]:4:1)
   fimpara
   escreval("Média Geral das Turmas: ", mediaGeral:4:1)
   
   // Analisando turmas acima da média
   escreval("")
   escreval("🏆 TURMAS DESTAQUE:")
   para i de 1 ate 3 faca
      se (medias[i] > mediaGeral) entao
         escreval("- Turma ", i, " (média ", medias[i]:4:1, ")")
      fimse
   fimpara
fimalgoritmo

🏋️ Exercícios Propostos

  1. Crie um algoritmo que preencha uma matriz 4x4 com números aleatórios e encontre o maior e o menor valor.
  2. Implemente um programa que leia duas matrizes 2x2 e mostre a matriz resultante da multiplicação.
  3. Desenvolva um algoritmo que leia um vetor de 10 nomes e ordene-os em ordem alfabética.
  4. Crie um programa que simule um campeonato com 4 times, onde cada time joga contra todos os outros, armazenando os resultados em uma matriz.
  5. Implemente um algoritmo para um sistema de reservas de teatro com 5 fileiras e 10 cadeiras por fileira.
