🎯 Projeto Final

🍕 DeliveryApp - Aplicativo de Entrega

Desenvolvimento completo de aplicativo mobile com todas as funcionalidades de um app real de delivery

  • 🔐 Autenticação: Login, registro, perfil
  • 🛒 E-commerce: Catálogo, carrinho, checkout
  • 📍 Geolocalização: Endereços e rastreamento
  • 💳 Pagamentos: Múltiplas formas de pagamento
  • 📱 Notificações: Push notifications
  • ⭐ Avaliações: Sistema de reviews

🏗️ Arquitetura do Projeto

📱 Frontend (React Native + Expo)

  • Screens: Login, Home, Product, Cart, Profile
  • Navigation: Stack + Tab Navigation
  • State Management: Context API + useReducer
  • Components: Reutilizáveis e modulares
  • Styling: StyleSheet + responsivo

☁️ Backend (Firebase)

  • Authentication: Email/password, Google
  • Firestore: Users, Products, Orders
  • Storage: Imagens de produtos
  • Functions: Serverless functions
  • Hosting: Deploy da admin web

📱 Principais Telas

🔐 Autenticação

  • Splash Screen
  • Login/Register
  • Esqueci senha
  • Onboarding

🛒 E-commerce

  • Lista de restaurantes
  • Menu de produtos
  • Detalhes do produto
  • Carrinho de compras

📍 Checkout

  • Endereço de entrega
  • Forma de pagamento
  • Resumo do pedido
  • Rastreamento

🔧 Setup do Projeto

📦 Criação do Projeto

# Criar projeto Expo
npx create-expo-app DeliveryApp
cd DeliveryApp

# Instalar dependências
npm install @react-navigation/native
npm install @react-navigation/stack
npm install @react-navigation/bottom-tabs
npm install react-native-screens
npm install react-native-safe-area-context
npm install firebase
npm install @expo/vector-icons
npm install expo-location
npm install expo-notifications

🔥 Configuração Firebase

# firebase.js
import { initializeApp } from 'firebase/app';
import { getAuth } from 'firebase/auth';
import { getFirestore } from 'firebase/firestore';
import { getStorage } from 'firebase/storage';

const firebaseConfig = {
  // Suas credenciais do Firebase
};

const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);
export const db = getFirestore(app);
export const storage = getStorage(app);

🧪 Implementação Prática

⏰ Cronograma de Desenvolvimento (4 horas)

  • Hora 1: Setup, navegação e autenticação básica
  • Hora 2: Telas principais e integração Firebase
  • Hora 3: Funcionalidades do carrinho e checkout
  • Hora 4: Finalização, testes e apresentação

🎯 Metodologia: Desenvolvimento Ágil

Trabalho em equipes de 3-4 pessoas, com roles definidos:

  • Frontend Developer: Telas e componentes
  • Backend Developer: Firebase integration
  • UI/UX Designer: Design e usabilidade
  • QA Tester: Testes e validação

📊 Critérios de Avaliação

🎯 Funcionalidade (40%)

  • Autenticação: Login/register funcionais
  • Navegação: Fluxo completo do app
  • CRUD: Operações com banco de dados
  • Integração: APIs e serviços externos

🎨 Interface (30%)

  • Design: Interface atrativa e intuitiva
  • Responsividade: Adaptação a diferentes telas
  • UX: Facilidade de uso
  • Consistência: Padrões visuais

💻 Código (20%)

  • Estrutura: Organização e modularização
  • Boas práticas: Clean code principles
  • Performance: Otimização e eficiência
  • Comentários: Documentação do código

🎤 Apresentação (10%)

  • Demonstração: Showcase das features
  • Explicação técnica: Decisões arquiteturais
  • Trabalho em equipe: Colaboração efetiva
  • Inovação: Funcionalidades extras
