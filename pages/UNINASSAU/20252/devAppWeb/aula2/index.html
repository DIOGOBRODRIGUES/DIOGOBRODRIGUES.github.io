📝 O que é HTML?

HTML (HyperText Markup Language)

É a linguagem de marcação padrão para criar páginas web.

Características:

  • Linguagem de marcação (não programação)
  • Utiliza tags para estruturar conteúdo
  • Interpretada pelos navegadores
  • Padrão W3C (World Wide Web Consortium)

🏗️ Estrutura Básica


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <h1>Minha Primeira Página</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

🔍 Anatomia de uma Tag HTML


<tagname attribute="value">Conteúdo</tagname>

Componentes:

  • Tag de abertura: <tagname>
  • Atributos: attribute="value"
  • Conteúdo: texto entre as tags
  • Tag de fechamento: </tagname>

📝 Listas HTML

📋 Tipos de Listas

Lista Ordenada


<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>
  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada


<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>
  • Item A
  • Item B
  • Item C

Lista de Definição


<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação</dd>
  <dt>CSS</dt>
  <dd>Folha de estilo</dd>
</dl>

✅ Validação HTML5


<form>
  <!-- Campo obrigatório -->
  <input type="text" required>
  
  <!-- Padrão com regex -->
  <input type="text" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" 
         placeholder="123-456-7890">
  
  <!-- Limites numéricos -->
  <input type="number" min="1" max="100">
  
  <!-- Tamanho do texto -->
  <input type="text" minlength="3" maxlength="20">
  
  <!-- Validação customizada -->
  <input type="email" title="Digite um email válido">
</form>

🎯 HTML Semântico

Utilizando tags com significado para melhor estrutura e acessibilidade

🏗️ Tags Semânticas HTML5

Estrutura da Página

  • <header> - Cabeçalho
  • <nav> - Navegação
  • <main> - Conteúdo principal
  • <article> - Artigo independente
  • <section> - Seção de conteúdo
  • <aside> - Conteúdo lateral
  • <footer> - Rodapé

Conteúdo

  • <figure> - Figura com legenda
  • <figcaption> - Legenda
  • <time> - Data/hora
  • <mark> - Texto destacado
  • <details> - Detalhes expansíveis
  • <summary> - Resumo
  • <address> - Informações de contato

📐 Exemplo de Estrutura Semântica


<!DOCTYPE html>
<html lang="pt-BR">
<body>
  <header>
    <h1>Meu Site</h1>
    <nav>
      <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#sobre">Sobre</a></li>
      </ul>
    </nav>
  </header>
  
  <main>
    <article>
      <h2>Título do Artigo</h2>
      <p>Conteúdo do artigo...</p>
    </article>
    
    <aside>
      <h3>Links Relacionados</h3>
      <ul><li><a href="#">Link 1</a></li></ul>
    </aside>
  </main>
  
  <footer>
    <p>&copy; 2025 Meu Site</p>
  </footer>
</body>
</html>

♿ Acessibilidade Web

Tornando o conteúdo acessível para todos os usuários

🎯 Princípios WCAG

🔍 Perceptível

  • Alt text em imagens
  • Contraste adequado
  • Legendas em vídeos

⌨️ Operável

  • Navegação por teclado
  • Tempo suficiente
  • Sem epilepsia

🧠 Compreensível

  • Texto legível
  • Funcionamento previsível
  • Ajuda na entrada

🔧 Robusto

  • Compatível com tecnologias assistivas
  • Código válido
  • Adaptável

🛠️ Técnicas de Acessibilidade


<!-- Texto alternativo -->
<img src="grafico.jpg" alt="Gráfico mostrando aumento de vendas em 2025">

<!-- Labels para formulários -->
<label for="nome">Nome completo:</label>
<input type="text" id="nome" name="nome" required>

<!-- Hierarquia de títulos -->
<h1>Título Principal</h1>
  <h2>Seção</h2>
    <h3>Subseção</h3>

<!-- Link descritivo -->
<a href="relatorio.pdf">Baixar relatório anual (PDF, 2MB)</a>

<!-- Atributos ARIA -->
<button aria-label="Fechar janela">×</button>
<div role="alert" aria-live="polite">Mensagem importante</div>

  1. Abrir Eclipse IDE
  2. File → New → Dynamic Web Project

✨ Boas Práticas HTML

✅ Faça

  • Use DOCTYPE html
  • Defina lang no html
  • Use UTF-8 charset
  • Tags em minúsculas
  • Feche todas as tags
  • Use semântica apropriada
  • Alt text em imagens
  • Labels em formulários

❌ Evite

  • Tags obsoletas (<center>, <font>)
  • Inline styles
  • IDs duplicados
  • Div para tudo
  • Tabelas para layout
  • Texto em imagens
  • Links vagos ("clique aqui")
  • Hierarquia de títulos incorreta
