🚀 Projeto

Sistema de Cadastro de Usuários

Desenvolver uma aplicação web completa que permita:

  • Cadastrar novos usuários
  • Listar usuários cadastrados
  • Editar informações
  • Excluir registros
  • Buscar usuários

🎯 Funcionalidades Principais

✅ CRUD Completo

  • Create - Criar novos registros
  • Read - Ler/Listar registros
  • Update - Atualizar registros
  • Delete - Deletar registros

📱 Interface Responsiva

Desktop

Layout completo com sidebar

Mobile

Menu hambúrguer e cards

📄 Campos do Formulário

  • Email (obrigatório, formato válido)
  • Telefone (obrigatório, máscara (99) 99999-9999)

  • Endereço (obrigatório)

  • Profissão (opcional)

🔍 Sistema de Busca


function searchUsers(query) {
    const users = userManager.getUsers();
    return users.filter(user => 
        user.name.toLowerCase().includes(query.toLowerCase()) ||
        user.email.toLowerCase().includes(query.toLowerCase()) ||
        user.city.toLowerCase().includes(query.toLowerCase())
    );
}

// Implementar busca em tempo real
document.getElementById('search-input').addEventListener('input', function(e) {
    const query = e.target.value;
    const filteredUsers = searchUsers(query);
    renderUserTable(filteredUsers);
});

❓ Perguntas Frequentes

🤔 Dúvidas Comuns

Q: Posso usar bibliotecas externas?

R: Sim, mas o foco deve ser em JavaScript vanilla. Bootstrap e jQuery são permitidos.

Q: Precisa funcionar em todos os navegadores?

R: Foque nos navegadores modernos (Chrome, Firefox, Safari, Edge).

Q: Como faço para persistir os dados?

R: Use localStorage do navegador. Não é necessário backend.
