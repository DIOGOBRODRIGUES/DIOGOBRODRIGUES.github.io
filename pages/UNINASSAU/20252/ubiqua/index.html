🎯 O que é Thunkable?

Plataforma de desenvolvimento mobile no-code

  • ✅ Crie apps iOS e Android simultaneamente
  • ✅ Interface visual drag-and-drop
  • ✅ Programação por blocos (similar ao Scratch)
  • ✅ Componentes prontos e customizáveis
  • ✅ Teste em tempo real no smartphone

⚙️ Configuração Inicial

Passo 1: Criar Conta

  1. Acesse thunkable.com
  2. Clique em "Sign Up"
  3. Use e-mail ou conta Google
  4. Confirme o e-mail

Passo 2: App Thunkable Live

  1. 📱 Baixe "Thunkable" na Play Store/App Store
  2. Faça login com a mesma conta
  3. Use para testar em tempo real

🖥️ Interface do Thunkable

1. Viewer (Visualizador) - Preview do app
2. Components (Componentes) - Elementos de UI
3. Properties (Propriedades) - Configurações visuais
4. Blocks (Blocos) - Lógica de programação

🎮 Game Design: Asteroids

Conceito do Jogo

🚀 Uma nave espacial deve desviar de asteroides que caem

  • Nave controlada por toque/acelerômetro
  • Asteroides caem de cima para baixo
  • Colisão = Game Over
  • Pontuação por tempo sobrevivido
  • Dificuldade progressiva

⚙️ Mecânicas Principais

🎯 Movimentação
  • Toque na tela
  • Arrastar nave
  • Limites da tela
☄️ Asteroides
  • Spawn aleatório
  • Velocidade variável
  • Reciclagem ao sair
💥 Colisão
  • Detecção de overlap
  • Som/vibração
  • Game Over
📊 Sistema
  • Score/Timer
  • High Score
  • Restart

✨ Melhorias Visuais

🎨 Efeitos
  • Partículas na colisão
  • Animação da nave
  • Trail de movimento
🔊 Sons
  • Background music
  • Som de explosão
  • Som de pontos
🌟 Extras
  • Power-ups
  • Diferentes tipos de asteroides
  • Parallax background
📱 UX
  • Feedback tátil
  • Tela de pausa
  • Tutorial inicial

📱 Testar no Dispositivo

Teste em Tempo Real:

  1. Abra o app Thunkable no celular
  2. Faça login com a mesma conta
  3. Clique em "Test" no Thunkable web
  4. Escaneie o QR Code que aparece
  5. O jogo abrirá no seu dispositivo!

🐛 Dicas de Debug:

  • Use blocos "Alert" para verificar valores
  • Teste incrementalmente
  • Verifique limites de tela

⚡ Otimização de Performance

✅ Boas Práticas
  • Timer interval ≥ 30ms
  • Imagens otimizadas (PNG/WebP)
  • Evite muitos componentes
  • Reutilize asteroides
❌ Evite
  • Timers muito rápidos
  • Imagens muito grandes
  • Loops infinitos
  • Muitas animações simultâneas

🔧 Pool de Objetos:


// Ao invés de criar/destruir, recicle
IF asteroid.Y > Screen1.Height THEN
  SET asteroid.Y to -50  // Reposiciona
  SET asteroid.X to random(0, Screen1.Width)
END IF

🚀 Publicar o App

Opções de Publicação:

📱 Android (APK)
  1. Clique em "Publish"
  2. Escolha "Android"
  3. Preencha detalhes
  4. Download do APK
  5. Instale no dispositivo
🍎 iOS (TestFlight)
  1. Requer conta Apple Developer
  2. Clique em "Publish"
  3. Escolha "iOS"
  4. Siga processo de review
  5. Distribua via TestFlight

💡 Android é mais fácil para começar!

🎯 Desafios Extras

🌟 Nível Fácil
  • Adicione efeito de vibração na colisão

  • Diferentes tamanhos de asteroides
  • Sistema de vidas (3 colisões)
💀 Nível Difícil
  • Controle por giroscópio
  • Multiplayer local
  • Leaderboard online
