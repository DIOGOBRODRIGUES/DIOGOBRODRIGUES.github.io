🚀 Mãos na Massa: Criando Nossa Primeira App

📝 Passo a Passo

1. Abrir terminal/prompt de comando

2. Navegar para pasta de projetos:

cd Desktop (ou onde preferir)

3. Criar aplicação React:

npx create-react-app minha-primeira-app

4. Entrar na pasta:

cd minha-primeira-app

5. Iniciar servidor de desenvolvimento:

npm start

⚛️ App.js: O Coração da Aplicação

import './App.css';

function App() {
  return (

      <header className="App-header">

🧩 JSX: Seu Primeiro Contato

💡 O que é JSX?

🧩 JSX = JavaScript + XML

Escrever HTML dentro do JavaScript

❌ JavaScript Tradicional

const element = 
  React.createElement(
    'h1', 
    null, 
    'Olá, Mundo!'
  );

✅ JSX

const element = 
  <h1>Olá, Mundo!</h1>;

🚨 Regras do JSX

✅ Pode fazer

  • Usar JavaScript dentro de {}
  • className em vez de class
  • onClick em vez de onclick
  • Componentes com letra maiúscula

❌ Não pode fazer

  • Retornar múltiplos elementos sem wrapper
  • Usar palavras reservadas do JS
  • Esquecer de fechar tags
  • Misturar HTML e JSX

🎯 Exemplo Prático

function MeuComponente() {
  const nome = "João";
  const idade = 25;

  return (
    <div>
      <h1>Olá, {nome}! 👋</h1>
      <p>Você tem {idade} anos</p>
      <button onClick={() => alert('Clicou!')}>
        Clique aqui
      </button>
    </div>
  );
}

🔥 Isso é puro poder do JSX!

🎯 Primeira Modificação: Personalizando sua App

✏️ Desafio Rápido

Vamos modificar o arquivo src/App.js:

function App() {
  const meuNome = "SEU_NOME_AQUI";
  const minhaIdade = SUA_IDADE;

  return (
    <div className="App">
      <header className="App-header">
        <h1>🚀 Minha Primeira App React</h1>
        <p>Olá! Eu sou {meuNome} e tenho {minhaIdade} anos</p>
        <p>🎯 Estou aprendendo React na Nassau!</p>
        <button onClick={() => alert('React é incrível! 🎉')}>
          Clique em mim!
        </button>
      </header>
    </div>
  );
}

🐛 Troubleshooting: Problemas Comuns

❌ Erros Mais Frequentes

🚨 "npm not found"

Solução: Reinstalar Node.js

🔗 nodejs.org

🚨 "Port 3000 is already in use"

Solução: Usar porta diferente

💻 npm start -- --port 3001

🚨 "Syntax Error"

Solução: Verificar JSX

Todas as tags fechadas?

🚨 "Module not found"

Solução: npm install

Dependências corrompidas

🔧 Comandos de Emergência

🧹 Limpar cache do npm:

npm cache clean --force

🔄 Reinstalar dependências:

rm -rf node_modules package-lock.json

npm install

🆘 Quando tudo falhar:

npx create-react-app novo-projeto
