StreamFlix

Projeto da Disciplina de Frameworks Front-End

Desenvolva uma aplicação web de streaming inspirada na Netflix utilizando React.js. Este projeto aplicará todos os conceitos fundamentais aprendidos durante o curso.

Prazo de Entrega

09 de Outubro de 2025 - Faltam dias

Visão Geral do Projeto

O projeto consiste no desenvolvimento de uma aplicação web front-end inspirada na Netflix, utilizando React.js como framework principal. O objetivo é aplicar os conceitos aprendidos em aula para criar uma interface moderna, responsiva e funcional.

Requisitos Funcionais

Página Inicial

  • Banner principal com filme/série em destaque
  • Carrosséis horizontais por categoria (Ação, Drama, Comédia, etc.)
  • Header com logo, menu de navegação e busca
  • Footer com informações da plataforma

Detalhes do Conteúdo

  • Página individual para cada filme/série
  • Exibição de poster, sinopse, elenco e ano
  • Botões "Assistir" e "Minha Lista"
  • Conteúdos relacionados/similares

Minha Lista

  • Adicionar/remover filmes da lista pessoal
  • Página dedicada aos favoritos
  • Persistência usando localStorage
  • Contador de itens na lista

Categorias

  • Pelo menos 5 categorias diferentes
  • Navegação entre categorias
  • Mínimo 8 títulos por categoria
  • Indicadores visuais da categoria ativa

Responsividade

  • Layout adaptável para desktop, tablet e mobile
  • Carrosséis funcionais em todos os dispositivos
  • Menu mobile com hamburger
  • Teste em pelo menos 3 resoluções diferentes

Requisitos Técnicos

React.js

  • Usar Create React App ou Vite
  • Componentes funcionais com Hooks
  • Mínimo 8 componentes personalizados
  • useState e useEffect obrigatórios

Estrutura

  • React Router para navegação
  • Pelo menos 4 rotas diferentes
  • Componentes organizados em pastas
  • Separação de responsabilidades

Estilização

  • CSS Modules ou Styled Components
  • Design similar à Netflix
  • Animações e transições suaves
  • Cores escuras como tema principal

Dados

  • JSON local ou API externa (TMDb recomendado)
  • Mínimo 40 filmes/séries
  • Dados realistas com posters

Critérios de Entrega

Repositório GitHub

  • Código-fonte completo no GitHub
  • README.md com instruções de instalação
  • Commits organizados e descritivos

Deploy

  • Aplicação publicada (Vercel, Netlify ou GitHub Pages)
  • Link funcionando no dia da apresentação
  • Performance otimizada
  • Build sem erros ou warnings

Apresentação

  • Demo ao vivo da aplicação (5-7 minutos)
  • Explicação do código principal
  • Demonstração das funcionalidades
  • Responder perguntas técnicas
