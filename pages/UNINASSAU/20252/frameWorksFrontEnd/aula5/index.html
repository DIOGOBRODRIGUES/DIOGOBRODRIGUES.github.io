� Tipos de Props

� Props podem ser qualquer coisa:

  • 🔤 Texto: nome, descrição
  • 🔢 Números: idade, preço, quantidade
  • ✅ Boolean: ativo/inativo, visível/oculto
  • 📋 Arrays: lista de itens
  • 🎛️ Funções: ações para botões

function ProductCard({ 
  name,          // 🔤 String
  price,         // 🔢 Number  
  inStock,       // ✅ Boolean
  tags,          // 📋 Array
  onBuy          // 🎛️ Function
}) {
  return (
    <div className="product">
      <h3>{name}</h3>
      <p>R$ {price}</p>
      
      {inStock ? (
        <span className="available">✅ Disponível</span>
      ) : (
        <span className="unavailable">❌ Esgotado</span>
      )}
      
      <div>
        {tags.map(tag => (
          <span key={tag} className="tag">{tag}</span>
        ))}
      </div>
      
      <button onClick={onBuy}>Comprar</button>
    </div>
  );
}

🎯 Demonstração: Botão Interativo


// 🏠 Componente Pai
function Counter() {
  const [count, setCount] = useState(0);

  // 🔔 Função que o filho pode "tocar"
  const handleIncrement = () => {
    setCount(count + 1);
    alert(`Contou ${count + 1} vezes!`);
  };

  return (
    <div>
      <h2>Contador: {count}</h2>
      <CounterButton 
        onPress={handleIncrement}
        label="Clique aqui!"
      />
    </div>
  );
}

// 🚪 Componente Filho
function CounterButton({ onPress, label }) {
  return (
    <button 
      onClick={onPress}  // 🔔 "Toca a campainha"
      className="big-button"
    >
      {label}
    </button>
  );
}

🎮 Exemplo Completo: Mini Loja

🛍️ Vamos criar uma loja simples!

O que vamos fazer:

  • 💰 Mostrar produtos com preços
  • 🛒 Adicionar itens ao carrinho
  • 🧮 Calcular total automático

🏪 Componente da Loja (Pai)


function Shop() {
  const [cart, setCart] = useState([]);
  
  const products = [
    { id: 1, name: "🍕 Pizza", price: 25.90 },
    { id: 2, name: "🍔 Burger", price: 18.50 },
    { id: 3, name: "🥤 Refrigerante", price: 5.00 }
  ];

  const addToCart = (product) => {
    setCart([...cart, product]);
  };

  const total = cart.reduce((sum, item) => sum + item.price, 0);

  return (
    <div>
      <h1>🛍️ Mini Loja</h1>
      
      <div className="products">
        {products.map(product => (
          <ProductCard
            key={product.id}
            name={product.name}
            price={product.price}
            onBuy={() => addToCart(product)}
          />
        ))}
      </div>
      
      <div className="cart">
        <h3>🛒 Carrinho ({cart.length} itens)</h3>
        <p>💰 Total: R$ {total.toFixed(2)}</p>
      </div>
    </div>
  );
}

⚖️ Regras Importantes das Props

✅ Pode Fazer

  • � Passar qualquer tipo de dado
  • 🔄 Passar funções como props
  • 📋 Usar arrays e objetos
  • 🎯 Usar destructuring

❌ Não Pode Fazer

  • 🚫 Modificar props no componente filho
  • 🚫 Props são somente leitura
  • 🚫 Filho não pode alterar dados do pai diretamente

// ❌ ERRADO - Não modifique props!
function BadComponent({ user }) {
  user.name = "Novo nome"; // 🚫 Não faça isso!
  return <h1>{user.name}</h1>;
}

// ✅ CORRETO - Props são apenas para leitura
function GoodComponent({ user }) {
  return <h1>{user.name}</h1>; // ✅ Apenas use!
}

🔄 Props vs State

🥊 Battle: Props vs State

Característica Props State
🔄 Mutabilidade Read-only Mutável
📍 Origem Componente pai Próprio componente
🎯 Propósito Configuração Dados dinâmicos
⚡ Re-render Quando pai muda Quando setState

🎯 Quando Usar Cada Um?

📤 Use Props quando:

  • 🔧 Configurar componente
  • 📊 Passar dados do pai
  • 🎨 Personalizar aparência
  • ⚡ Dados não mudam no componente

🔄 Use State quando:

  • 📝 Dados mudam com interação
  • 🎮 Controlar UI dinamicamente
  • 📊 Armazenar dados temporários
  • 🔄 Atualizar após eventos

// Componente Pai
function App() {
  const [message, setMessage] = useState('');

  const handleChildMessage = (childMessage) => {
    setMessage(childMessage);
    alert(`Filho disse: ${childMessage}`);
  };

  return (
    <div>
      <h2>Mensagem do filho: {message}</h2>
      <ChildComponent onSendMessage={handleChildMessage} />
    </div>
  );
}

// Componente Filho
function ChildComponent({ onSendMessage }) {
  const handleClick = () => {
    onSendMessage('Oi pai! 👋');
  };

  return <button onClick={handleClick}>Falar com o Pai</button>;
}

}

🎯 Projeto Prático: Todo List

📋 Estrutura da Todo List


function TodoApp() {
  const [todos, setTodos] = useState([]);

  const addTodo = (text) => {
    const newTodo = {
      id: Date.now(),
      text: text,
      completed: false
    };
    setTodos([...todos, newTodo]);
  };

  const toggleTodo = (id) => {
    setTodos(todos.map(todo => 
      todo.id === id 
        ? { ...todo, completed: !todo.completed }
        : todo
    ));
  };

  return (
    <div>
      <TodoForm onAdd={addTodo} />
      <TodoList todos={todos} onToggle={toggleTodo} />
    </div>
  );
}

📝 Componente TodoForm


function TodoForm({ onAdd }) {
  const [inputValue, setInputValue] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    if (inputValue.trim()) {
      onAdd(inputValue);
      setInputValue('');
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
        placeholder="Adicionar nova tarefa..."
      />
      <button type="submit">➕ Adicionar</button>
    </form>
  );
}

📋 Componente TodoList


function TodoList({ todos, onToggle }) {
  return (
    <ul className="todo-list">
      {todos.map(todo => (
        <TodoItem 
          key={todo.id}
          todo={todo}
          onToggle={onToggle}
        />
      ))}
    </ul>
  );
}

function TodoItem({ todo, onToggle }) {
  return (
    <li className={todo.completed ? 'completed' : ''}>
      <input
        type="checkbox"
        checked={todo.completed}
        onChange={() => onToggle(todo.id)}
      />
      <span>{todo.text}</span>
    </li>
  );
}

💡 Dicas e Boas Práticas

🎯 10 Regras de Ouro das Props

✅ Faça:

  • 📝 Use nomes descritivos
  • 🧹 Mantenha props simples
  • 🔍 Valide com PropTypes
  • 🎯 Use destructuring
  • 📦 Agrupe props relacionadas

❌ Evite:

  • 🔄 Mudar props diretamente
  • 📊 Props com muitos níveis
  • 🎭 Props só para passar adiante
  • 🔧 Lógica complexa nas props
  • 📋 Muitas props (>5-7)

🚀 Pattern: Compound Components


// Padrão avançado - componentes que trabalham juntos
function Card({ children }) {
  return <div className="card">{children}</div>;
}

Card.Header = function CardHeader({ children }) {
  return <div className="card-header">{children}</div>;
};

Card.Body = function CardBody({ children }) {
  return <div className="card-body">{children}</div>;
};

// Uso
<Card>
  <Card.Header>Título</Card.Header>
  <Card.Body>Conteúdo</Card.Body>
</Card>
