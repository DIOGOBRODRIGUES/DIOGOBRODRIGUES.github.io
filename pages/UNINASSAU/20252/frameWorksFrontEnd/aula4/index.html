🔄 O que é Estado (State)?

🎯 Definição:

Estado é dados que um componente "lembra" e que podem mudar ao longo do tempo

📝 Exemplos de Estado:

  • 🔢 Contador de cliques
  • 📝 Texto digitado em um input
  • 👁️ Mostrar/ocultar um elemento
  • 🎨 Tema claro/escuro
  • 📋 Lista de tarefas
  • 👤 Dados do usuário logado

⚡ Características:

  • ✅ Dados privados do componente
  • ✅ Quando muda, o componente re-renderiza
  • ✅ Inicializado uma vez quando componente nasce

⚖️ Componente Sem Estado vs Com Estado

function Contador() {

🪝 Hook useState - Sua Nova Ferramenta

📋 Sintaxe:

import React, { useState } from 'react';

function MeuComponente() {
  const [estado, setEstado] = useState(valorInicial);
  
  // estado = valor atual
  // setEstado = função para mudar o valor
  // valorInicial = valor quando componente nasce
}

🔍 Decompondo:

  • useState = Hook que cria estado
  • [estado, setEstado] = Destructuring Array
  • estado = Variável com valor atual
  • setEstado = Função para atualizar
  • valorInicial = Valor inicial do estado

💫 Primeiro Exemplo: Contador Simples

import React, { useState } from 'react';

function Contador() {
  // Criando estado que começa com 0
  const [contador, setContador] = useState(0);

  // Função para aumentar
  const aumentar = () => {
    setContador(contador + 1);
  };

  // Função para diminuir
  const diminuir = () => {
    setContador(contador - 1);
  };

  return (
    <div style={{ textAlign: 'center', margin: '20px' }}>
      <h2>Contador: {contador}</h2>
      <button onClick={diminuir}> - </button>
      <button onClick={aumentar}> + </button>
    </div>
  );
}

export default Contador;

🖱️ Eventos em React

🎯 O que são Eventos?

Ações que o usuário faz na interface que nosso código pode "escutar"

📝 Eventos Mais Comuns:

  • onClick - Clique do mouse
  • onChange - Mudança em input
  • onSubmit - Envio de formulário
  • onFocus - Foco no elemento
  • onBlur - Perda de foco
  • onMouseOver - Mouse em cima
  • onKeyDown - Tecla pressionada
  • onDoubleClick - Duplo clique

⚡ Diferença do HTML:

  • HTML: onclick="minhaFuncao()"
  • React: onClick={minhaFuncao}

🎮 Exemplo: Botão Interativo

function BotaoInterativo() {
  const [clicado, setClicado] = useState(false);
  const [contador, setContador] = useState(0);

  const handleClick = () => {
    setClicado(true);
    setContador(contador + 1);
    console.log('Botão foi clicado!');
  };

  const handleMouseOver = () => {
    console.log('Mouse passou por cima!');
  };

  return (
    <div>
      <button 
        onClick={handleClick}
        onMouseOver={handleMouseOver}
        style={{
          backgroundColor: clicado ? 'green' : 'blue',
          color: 'white',
          padding: '10px 20px',
          border: 'none',
          borderRadius: '5px'
        }}
      >
        Clicado {contador} vezes
      </button>
      
      {clicado && <p>Obrigado por clicar! 🎉</p>}
    </div>
  );
}

📝 Input Controlado - Formulários em React

function FormularioSimples() {
  const [nome, setNome] = useState('');
  const [email, setEmail] = useState('');

  const handleNomeChange = (event) => {
    setNome(event.target.value);
  };

  const handleEmailChange = (event) => {
    setEmail(event.target.value);
  };

  const handleSubmit = (event) => {
    event.preventDefault(); // Impede reload da página
    alert(`Nome: ${nome}, Email: ${email}`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <div>
        <label>Nome:</label>
        <input 
          type="text" 
          value={nome}
          onChange={handleNomeChange}
          placeholder="Digite seu nome"
        />
      </div>
      
      <div>
        <label>Email:</label>
        <input 
          type="email" 
          value={email}
          onChange={handleEmailChange}
          placeholder="Digite seu email"
        />
      </div>
      
      <button type="submit">Enviar</button>
      
      <div>
        <h3>Preview:</h3>
        <p>Nome: {nome}</p>
        <p>Email: {email}</p>
      </div>
    </form>
  );
}

👁️ Exemplo: Toggle de Visibilidade

function ToggleVisibilidade() {
  const [mostrar, setMostrar] = useState(false);

  const alternarVisibilidade = () => {
    setMostrar(!mostrar); // Inverte o valor (true vira false, false vira true)
  };

  return (
    <div>
      <button onClick={alternarVisibilidade}>
        {mostrar ? 'Ocultar' : 'Mostrar'} Conteúdo
      </button>
      
      {mostrar && (
        <div style={{
          backgroundColor: '#f0f0f0',
          padding: '20px',
          margin: '10px 0',
          borderRadius: '5px',
          color: 'black'
        }}>
          <h3>Conteúdo Secreto! 🎭</h3>
          <p>Este conteúdo só aparece quando você clica no botão!</p>
          <p>Estado atual: {mostrar ? 'Visível' : 'Oculto'}</p>
        </div>
      )}
    </div>
  );
}

📋 Exemplo: Lista de Tarefas

function ListaTarefas() {
  const [tarefas, setTarefas] = useState(['Estudar React', 'Fazer exercícios']);
  const [novaTarefa, setNovaTarefa] = useState('');

  const adicionarTarefa = () => {
    if (novaTarefa.trim() !== '') {
      setTarefas([...tarefas, novaTarefa]); // Spread operator
      setNovaTarefa(''); // Limpa o input
    }
  };

  const removerTarefa = (index) => {
    const novaLista = tarefas.filter((_, i) => i !== index);
    setTarefas(novaLista);
  };

  return (
    <div>
      <h2>Minhas Tarefas ({tarefas.length})</h2>
      
      <div>
        <input 
          type="text"
          value={novaTarefa}
          onChange={(e) => setNovaTarefa(e.target.value)}
          placeholder="Digite uma nova tarefa"
        />
        <button onClick={adicionarTarefa}>Adicionar</button>
      </div>
      
      <ul>
        {tarefas.map((tarefa, index) => (
          <li key={index}>
            {tarefa}
            <button onClick={() => removerTarefa(index)}>
              ❌
            </button>
          </li>
        ))}
      </ul>
    </div>
  );
}

✅ Boas Práticas com Estado e Eventos

🎯 Nomenclatura:

  • handle + Ação
  • handleClick
  • handleChange
  • handleSubmit

🔄 Atualizando Estado:

  • ✅ setCount(count + 1)
  • ✅ setItems([...items, newItem])
  • ❌ count++
  • ❌ items.push(newItem)

⚡ Performance:

  • Estado muda → componente re-renderiza
  • Evite estado desnecessário
  • Use callback quando possível

🎮 Eventos:

  • Use event.preventDefault() em forms
  • Arrow functions para parâmetros
  • onClick={() => funcao(param)}

🧮 Projeto: Calculadora Simples

function Calculadora() {
  const [display, setDisplay] = useState('0');
  const [operacao, setOperacao] = useState(null);
  const [valor, setValor] = useState(null);

  const inserirNumero = (numero) => {
    if (display === '0') {
      setDisplay(numero.toString());
    } else {
      setDisplay(display + numero.toString());
    }
  };

  const inserirOperacao = (op) => {
    setValor(parseFloat(display));
    setOperacao(op);
    setDisplay('0');
  };

  const calcular = () => {
    const atual = parseFloat(display);
    let resultado;
    
    switch (operacao) {
      case '+': resultado = valor + atual; break;
      case '-': resultado = valor - atual; break;
      case '*': resultado = valor * atual; break;
      case '/': resultado = valor / atual; break;
      default: return;
    }
    
    setDisplay(resultado.toString());
    setOperacao(null);
    setValor(null);
  };

  const limpar = () => {
    setDisplay('0');
    setOperacao(null);
    setValor(null);
  };

  return (
    <div style={{ textAlign: 'center', maxWidth: '300px', margin: '0 auto' }}>
      <div style={{ 
        backgroundColor: '#000', 
        color: '#0f0', 
        padding: '20px', 
        fontSize: '24px',
        fontFamily: 'monospace'
      }}>
        {display}
      </div>
      
      <div style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: '5px' }}>
        <button onClick={limpar}>C</button>
        <button onClick={() => inserirOperacao('/')}>&divide;</button>
        <button onClick={() => inserirOperacao('*')}&times;</button>
        <button onClick={() => inserirOperacao('-')}&minus;</button>
        
        <button onClick={() => inserirNumero(7)}>7</button>
        <button onClick={() => inserirNumero(8)}>8</button>
        <button onClick={() => inserirNumero(9)}>9</button>
        <button onClick={() => inserirOperacao('+')}&plus;</button>
        
        <button onClick={() => inserirNumero(4)}>4</button>
        <button onClick={() => inserirNumero(5)}>5</button>
        <button onClick={() => inserirNumero(6)}>6</button>
        <button onClick={calcular} rowSpan="2">=</button>
        
        <button onClick={() => inserirNumero(1)}>1</button>
        <button onClick={() => inserirNumero(2)}>2</button>
        <button onClick={() => inserirNumero(3)}>3</button>
        
        <button onClick={() => inserirNumero(0)} style={{ gridColumn: 'span 2' }}>0</button>
        <button onClick={() => inserirNumero('.')}>.</button>
      </div>
    </div>
  );
}

🎨 Projeto: Mudança de Tema

function MudancaTema() {
  const [temaEscuro, setTemaEscuro] = useState(false);

  const temas = {
    claro: {
      backgroundColor: '#ffffff',
      color: '#333333',
      buttonBg: '#007bff',
      buttonColor: '#ffffff'
    },
    escuro: {
      backgroundColor: '#333333',
      color: '#ffffff',
      buttonBg: '#ffc107',
      buttonColor: '#000000'
    }
  };

  const temaAtual = temaEscuro ? temas.escuro : temas.claro;

  const alternarTema = () => {
    setTemaEscuro(!temaEscuro);
  };

  return (
    <div style={{
      backgroundColor: temaAtual.backgroundColor,
      color: temaAtual.color,
      padding: '40px',
      minHeight: '400px',
      transition: 'all 0.3s ease'
    }}>
      <h2>🎨 Demonstração de Tema</h2>
      
      <button 
        onClick={alternarTema}
        style={{
          backgroundColor: temaAtual.buttonBg,
          color: temaAtual.buttonColor,
          border: 'none',
          padding: '10px 20px',
          borderRadius: '5px',
          cursor: 'pointer',
          fontSize: '16px'
        }}
      >
        {temaEscuro ? '☀️ Tema Claro' : '🌙 Tema Escuro'}
      </button>
      
      <div style={{ marginTop: '20px' }}>
        <h3>Conteúdo da Página</h3>
        <p>Este é um exemplo de como implementar mudança de tema usando estado.</p>
        <p>Tema atual: {temaEscuro ? 'Escuro' : 'Claro'}</p>
        
        <div style={{
          backgroundColor: temaEscuro ? '#555' : '#f0f0f0',
          padding: '20px',
          borderRadius: '5px',
          marginTop: '10px'
        }}>
          <h4>Card de Exemplo</h4>
          <p>O tema muda automaticamente todos os elementos!</p>
        </div>
      </div>
    </div>
  );
}

🚨 Erros Comuns e Como Evitar

❌ Erros Comuns:

// ❌ Modificar estado diretamente
count++;
items.push(newItem);

// ❌ Não usar função no onClick
onClick={handleClick()} // executa imediatamente

// ❌ Esquecer event.preventDefault
handleSubmit() { /* sem preventDefault */ }

// ❌ Estado inicial undefined
const [user, setUser] = useState();
user.name // ❌ Error!

✅ Forma Correta:

// ✅ Usar função set
setCount(count + 1);
setItems([...items, newItem]);

// ✅ Passar referência da função
onClick={handleClick}

// ✅ Prevenir comportamento padrão
const handleSubmit = (e) => {
  e.preventDefault();
}

// ✅ Estado inicial definido
const [user, setUser] = useState({
  name: '', email: ''
});
