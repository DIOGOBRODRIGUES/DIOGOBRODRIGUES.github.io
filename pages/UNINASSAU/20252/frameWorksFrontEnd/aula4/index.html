🔄 Conceito: O que é Estado?

📚 Definição

Estado é a "memória" de um componente - dados que ele guarda e que podem mudar durante sua vida útil.

🤖 No Mundo Real

  • 🧠 Sua memória (nome, idade, humor)
  • 📱 Bateria do celular (0% a 100%)
  • 🚦 Semáforo (vermelho, amarelo, verde)
  • 💧 Copo d'água (vazio, meio, cheio)

💻 No React

  • 🔢 Contador de likes
  • 📝 Texto digitado
  • 👁️ Modal aberto/fechado
  • 🎨 Tema claro/escuro

⚖️ Estático vs Dinâmico

🖼️ Analogia: Foto vs Filme

📸 Componente Estático = Foto

  • 🎯 Sempre igual
  • 📊 Dados fixos
  • 🚫 Não reage a interações
  • 🏷️ Exemplo: Logo, título
function Logo() {
  return <h1>Minha Empresa</h1>;
}

🎬 Componente Dinâmico = Filme

  • 🔄 Muda com o tempo
  • ⚡ Reage a eventos
  • 🎮 Interativo
  • 🏷️ Exemplo: Contador, formulário
function Contador() {
  const [numero, setNumero] = useState(0);
  return <button>{numero}</button>;
}

🪝 Hook useState - Sua Ferramenta Mágica

🎭 Analogia: Controle Remoto

O useState é como um controle remoto para seus dados!

📺 Controle Remoto

  • 📟 Display: Mostra canal atual
  • 🔼 Botão ↑: Muda para próximo canal
  • 🔽 Botão ↓: Volta canal anterior

⚛️ useState Hook

  • 📟 Estado: Guarda valor atual
  • 🔼 Setter: Função para alterar
  • 🎯 Inicial: Valor quando "liga"
Sintaxe:
const [valor, setValor] = useState(valorInicial);

🔬 Anatomia do useState

const [contador, setContador] = useState(0);

📊 contador

Variável que guarda o valor atual

🎯 Só leitura!

🔧 setContador

Função para alterar o valor

⚡ Causa re-render!

🌱 useState(0)

Hook que cria o estado

🎬 Valor inicial: 0

⚠️ Regra de Ouro: Nunca altere o estado diretamente! Sempre use a função setter.

🔄 Ciclo de Vida do Estado

🌰 1. Nascimento

useState(valorInicial)

Planta a semente

🌿 2. Crescimento

setEstado(novoValor)

Árvore cresce

🍃 3. Re-render

Componente atualiza

Folhas mudam

📝 Processo:

  1. 🎬 Inicialização: Estado nasce com valor inicial
  2. 🖱️ Evento: Usuário interage (clique, digitação)
  3. 🔧 Atualização: Função setter é chamada
  4. 🎨 Re-render: React atualiza a tela
  5. 🔄 Ciclo: Processo se repete

🖱️ Eventos: Escutando o Usuário

Eventos são como um garçom que está sempre atento aos pedidos dos clientes!

🍽️ Restaurante

  • 👂 Garçom escuta pedidos
  • 📝 Anota o que cliente quer
  • 🏃 Vai até a cozinha
  • 🍕 Traz a comida

💻 React

  • 👂 Event listener escuta
  • 📝 Recebe evento (clique, digitação)
  • 🏃 Executa função handler
  • 🎨 Atualiza interface

🎭 Principais Tipos de Eventos

🖱️ Mouse

  • onClick - Clique
  • onMouseOver - Mouse em cima
  • onMouseOut - Mouse saiu

⌨️ Teclado

  • onChange - Texto mudou
  • onKeyPress - Tecla pressionada
  • onFocus - Campo selecionado

📝 Formulário

  • onSubmit - Envio do form
  • onInput - Digitação
  • onBlur - Perdeu foco

📱 Touch (Mobile)

  • onTouchStart - Toque iniciou
  • onTouchMove - Deslizou
  • onTouchEnd - Soltou

                        }

✅ Boas Práticas com Estado

✅ Faça

  • 🎯 Use nomes descritivos
  • 🧪 Inicie com valor lógico
  • 🔧 Sempre use a função setter
  • 📦 Mantenha estados simples
  • 🎪 Um useState por dados

❌ Evite

  • 🚫 Alterar estado diretamente
  • 🌀 Estados muito complexos
  • 💾 Dados que não mudam
  • ⚡ Setters em render

"Se não muda, não é estado. Se muda, provavelmente é!"

✅ Boas Práticas com Estado e Eventos

🎯 Nomenclatura:

  • handle + Ação
  • handleClick
  • handleChange
  • handleSubmit

🔄 Atualizando Estado:

  • ✅ setCount(count + 1)
  • ✅ setItems([...items, newItem])
  • ❌ count++
  • ❌ items.push(newItem)

⚡ Performance:

  • Estado muda → componente re-renderiza
  • Evite estado desnecessário
  • Use callback quando possível

🎮 Eventos:

  • Use event.preventDefault() em forms
  • Arrow functions para parâmetros
  • onClick={() => funcao(param)}

🧮 Projeto: Calculadora Simples

function Calculadora() {
  const [display, setDisplay] = useState('0');
  const [operacao, setOperacao] = useState(null);
  const [valor, setValor] = useState(null);

  const inserirNumero = (numero) => {
    if (display === '0') {
      setDisplay(numero.toString());
    } else {
      setDisplay(display + numero.toString());
    }
  };

  const inserirOperacao = (op) => {
    setValor(parseFloat(display));
    setOperacao(op);
    setDisplay('0');
  };

  const calcular = () => {
    const atual = parseFloat(display);
    let resultado;
    
    switch (operacao) {
      case '+': resultado = valor + atual; break;
      case '-': resultado = valor - atual; break;
      case '*': resultado = valor * atual; break;
      case '/': resultado = valor / atual; break;
      default: return;
    }
    
    setDisplay(resultado.toString());
    setOperacao(null);
    setValor(null);
  };

  const limpar = () => {
    setDisplay('0');
    setOperacao(null);
    setValor(null);
  };

  return (
    <div style={{ textAlign: 'center', maxWidth: '300px', margin: '0 auto' }}>
      <div style={{ 
        backgroundColor: '#000', 
        color: '#0f0', 
        padding: '20px', 
        fontSize: '24px',
        fontFamily: 'monospace'
      }}>
        {display}
      </div>
      
      <div style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: '5px' }}>
        <button onClick={limpar}>C</button>
        <button onClick={() => inserirOperacao('/')}>&divide;</button>
        <button onClick={() => inserirOperacao('*')}&times;</button>
        <button onClick={() => inserirOperacao('-')}&minus;</button>
        
        <button onClick={() => inserirNumero(7)}>7</button>
        <button onClick={() => inserirNumero(8)}>8</button>
        <button onClick={() => inserirNumero(9)}>9</button>
        <button onClick={() => inserirOperacao('+')}&plus;</button>
        
        <button onClick={() => inserirNumero(4)}>4</button>
        <button onClick={() => inserirNumero(5)}>5</button>
        <button onClick={() => inserirNumero(6)}>6</button>
        <button onClick={calcular} rowSpan="2">=</button>
        
        <button onClick={() => inserirNumero(1)}>1</button>
        <button onClick={() => inserirNumero(2)}>2</button>
        <button onClick={() => inserirNumero(3)}>3</button>
        
        <button onClick={() => inserirNumero(0)} style={{ gridColumn: 'span 2' }}>0</button>
        <button onClick={() => inserirNumero('.')}>.</button>
      </div>
    </div>
  );
}

🎨 Projeto: Mudança de Tema

function MudancaTema() {
  const [temaEscuro, setTemaEscuro] = useState(false);

  const temas = {
    claro: {
      backgroundColor: '#ffffff',
      color: '#333333',
      buttonBg: '#007bff',
      buttonColor: '#ffffff'
    },
    escuro: {
      backgroundColor: '#333333',
      color: '#ffffff',
      buttonBg: '#ffc107',
      buttonColor: '#000000'
    }
  };

  const temaAtual = temaEscuro ? temas.escuro : temas.claro;

  const alternarTema = () => {
    setTemaEscuro(!temaEscuro);
  };

  return (
    <div style={{
      backgroundColor: temaAtual.backgroundColor,
      color: temaAtual.color,
      padding: '40px',
      minHeight: '400px',
      transition: 'all 0.3s ease'
    }}>
      <h2>🎨 Demonstração de Tema</h2>
      
      <button 
        onClick={alternarTema}
        style={{
          backgroundColor: temaAtual.buttonBg,
          color: temaAtual.buttonColor,
          border: 'none',
          padding: '10px 20px',
          borderRadius: '5px',
          cursor: 'pointer',
          fontSize: '16px'
        }}
      >
        {temaEscuro ? '☀️ Tema Claro' : '🌙 Tema Escuro'}
      </button>
      
      <div style={{ marginTop: '20px' }}>
        <h3>Conteúdo da Página</h3>
        <p>Este é um exemplo de como implementar mudança de tema usando estado.</p>
        <p>Tema atual: {temaEscuro ? 'Escuro' : 'Claro'}</p>
        
        <div style={{
          backgroundColor: temaEscuro ? '#555' : '#f0f0f0',
          padding: '20px',
          borderRadius: '5px',
          marginTop: '10px'
        }}>
          <h4>Card de Exemplo</h4>
          <p>O tema muda automaticamente todos os elementos!</p>
        </div>
      </div>
    </div>
  );
}

🚨 Erros Comuns e Como Evitar

❌ Erros Comuns:

// ❌ Modificar estado diretamente
count++;
items.push(newItem);

// ❌ Não usar função no onClick
onClick={handleClick()} // executa imediatamente

// ❌ Esquecer event.preventDefault
handleSubmit() { /* sem preventDefault */ }

// ❌ Estado inicial undefined
const [user, setUser] = useState();
user.name // ❌ Error!

✅ Forma Correta:

// ✅ Usar função set
setCount(count + 1);
setItems([...items, newItem]);

// ✅ Passar referência da função
onClick={handleClick}

// ✅ Prevenir comportamento padrão
const handleSubmit = (e) => {
  e.preventDefault();
}

// ✅ Estado inicial definido
const [user, setUser] = useState({
  name: '', email: ''
});
