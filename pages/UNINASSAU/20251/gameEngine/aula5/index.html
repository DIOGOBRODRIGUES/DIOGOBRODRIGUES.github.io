if Input.is_action_pressed("ui_accept"):

📢 Eventos de Entrada: O Que Acontece?

  • Teclado: Uma tecla foi pressionada? InputEventKey (pressed, keycode).
  • Mouse: Clique ou movimento? InputEventMouseButton, InputEventMouseMotion.
  • Gamepad: Botão ou analógico? InputEventJoypadButton, InputEventJoypadMotion.
  • Touch: Toque ou arraste? InputEventScreenTouch, InputEventScreenDrag.

🚶💨 6. Movimentação Básica: Colocando as Coisas para Andar!

Movimento com Rotação

  • Gire e mova como um carro! 🚗 
    
extends CharacterBody2D

var velocidade = 400
var velocidade_rotacao = 1.5

func _physics_process(delta):
  var entrada_movimento = Input.get_axis("down", "up")
  var direcao_rotacao = Input.get_axis("left", "right")
  velocity = transform.x * entrada_movimento * velocidade
  rotation += direcao_rotacao * velocidade_rotacao * delta
  move_and_slide()

Prática: Movimentação e Colisão 2D com Godot Engine

Objetivo: Implementar a movimentação de um personagem 2D e a detecção de colisões com objetos estáticos.

Passo 1: Criar um Novo Projeto

  1. Abra o Godot Engine.
  2. Clique em "Novo Projeto".
  3. Escolha um nome para o seu projeto e selecione um local para salvá-lo.
  4. No campo "Renderizador", escolha "Forward+ (Compatível)".
  5. Clique em "Criar e Editar".

Passo 2: Criar a Cena do Personagem

  1. Na aba "Cena", clique no botão "+" para criar um novo nó.
  2. Procure por "CharacterBody2D" e clique em "Criar".
  3. Nomeie este nó como "Player".
  4. Adicione um nó filho do tipo "Sprite2D" ao nó "Player".
  5. No Inspetor do nó "Sprite2D", procure pela propriedade "Textura". Clique em "[vazio]" e selecione "Criar Novo -> Atlas de Sprites" ou carregue sua imagem.

Passo 2: Criar a Cena do Personagem

  1. Adicione outro nó filho do tipo "CollisionShape2D" ao nó "Player".
  2. No Inspetor do nó "CollisionShape2D", procure pela propriedade "Forma". Clique em "[vazio]" e selecione "Criar Novo -> RetânguloShape2D".
  3. Ajuste o tamanho do retângulo de colisão para envolver o sprite do personagem na viewport.

Passo 3: Criar o Script do Personagem

  1. Selecione o nó "Player".
  2. Na aba "Inspetor", clique em "Script" e depois no botão "+" para criar um novo script.
  3. Mantenha as configurações padrão e clique em "Criar".
  4. Abra o script.
  5. Implemente o script para o personagem andar e pular.

Passo 4: Criar a Cena do Mundo

  1. Na aba "Cena", clique no botão "+" para criar uma nova cena.
  2. Escolha um nó raiz do tipo "Node2D" e clique em "Criar".
  3. Nomeie este nó como "World".
  4. Salve esta cena (Ctrl+S ou Cmd+S) com o nome "world.tscn".

Passo 5: Instanciar o Personagem

  1. Na aba "Cena" do nó "World", clique no botão "+" e selecione a aba "Cena".
  2. Procure e selecione a cena do seu personagem.
  3. Clique em "Criar".
  4. Arraste a instância do personagem para a posição inicial desejada na viewport.

Passo 6: Criar Objetos Estáticos para Colisão

  1. Na aba "Cena" do nó "World", clique no botão "+" para criar um novo nó.
  2. Procure por "StaticBody2D" e clique em "Criar".
  3. Adicione um nó filho do tipo "Sprite2D" ao nó "StaticBody2D". Carregue uma textura ou crie uma forma simples.
  4. Adicione um nó filho do tipo "CollisionShape2D" ao nó "StaticBody2D". Crie uma forma de colisão (ex: "RectangleShape2D") e ajuste o tamanho.

Passo 7: Criar Objetos Estáticos para Colisão

  1. Duplique o nó "StaticBody2D" algumas vezes (Ctrl+D ou Cmd+D).
  2. Posicione os objetos estáticos na cena para criar obstáculos ou um cenário.

Passo 8: Executar o Jogo

  1. Pressione F5 para executar o jogo.
  2. O Godot perguntará qual é a cena principal. Selecione a cena "world.tscn".
  3. Use as teclas de seta do teclado para mover o personagem.
  4. Observe que o personagem não atravessa os objetos estáticos.

Desafios Adicionais (Opcional)

  • Adicionar animações ao personagem com base na direção do movimento.
  • Implementar diferentes tipos de colisões (por exemplo, coletar itens).
  • Criar um cenário mais elaborado com diferentes formas de objetos estáticos.
  • Experimentar com as propriedades do CharacterBody2D e do CollisionShape2D.
