⚙️ Configuração e Primeiros Passos

  1. Acesse: thunkable.com
  2. Crie uma conta (gratuita)
  3. Escolha o tipo de projeto:
    • 📱 StageCanvas: Drag & Drop visual
    • 🧩 Snap to Place: Programação por blocos
  4. Instale o app Thunkable Live para testes

💡 Projeto Prático 1: Calculadora

🧮

Vamos criar uma calculadora simples!

Tempo estimado: 20 minutos

🎯 Objetivos da Calculadora

  • 🔢 Entrada de dois números
  • ➕ Soma simples
  • 📺 Exibir resultado
  • 🔄 Botão limpar
  • ✨ Interface básica e funcional

🏗️ Estrutura da Calculadora

  1. Campo 1: TextInput para o primeiro número
  2. Campo 2: TextInput para o segundo número
  3. Botão Somar: Button para calcular
  4. Resultado: Label para mostrar o resultado
  5. Botão Limpar: Button para limpar campos

🧩 Lógica Principal:

  • Obter valores dos TextInputs
  • Somar os dois números
  • Exibir resultado no Label
  • Limpar campos quando necessário

💾 Armazenamento de Dados

🗃️ Local Storage

  • 📱 Dados armazenados no dispositivo
  • 🔄 Persistem entre sessões
  • 🚀 Acesso rápido
  • 📊 Ideal para configurações

💾 Local DB

  • 🗄️ Banco de dados local
  • 📋 Listas e tabelas
  • 🔍 Busca e filtros
  • 📱 Para dados estruturados

💡 Projeto Prático 3: Lista de Tarefas

📝

To-Do List com persistência de dados!

Tempo estimado: 35 minutos

🎯 Funcionalidades

  • ➕ Adicionar nova tarefa
  • ✅ Marcar como concluída
  • 🗑️ Excluir tarefa
  • ✏️ Editar tarefa
  • 💾 Salvar dados localmente
  • 📊 Contador de tarefas

🏗️ Estrutura do App

🎨 Interface:

  • TextInput (nova tarefa)
  • Button (adicionar)
  • List Viewer (lista de tarefas)
  • Labels (contadores)

💾 Dados:

  • Local DB (tarefas)
  • Variáveis (lista atual)
  • Estados (concluída/pendente)

🧩 Lógica Principal:

  1. Carregar tarefas salvas na inicialização
  2. Adicionar/editar/excluir tarefas
  3. Atualizar interface em tempo real
  4. Salvar mudanças no banco local
