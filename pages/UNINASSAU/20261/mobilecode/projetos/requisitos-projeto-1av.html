Informações Gerais do Projeto Extra

Oportunidade de Pontuação Extra

Este projeto permite que você ganhe até 2.0 pontos na 1 AV que serão somados à sua nota final da disciplina. É uma excelente oportunidade para:

• Melhorar sua nota da 1ºAV
• Aplicar na prática todo o conteúdo estudado
• Desenvolver um portfólio profissional
• Demonstrar domínio avançado de React Native

Requisitos Obrigatórios do Projeto

Para receber os 2.0 pontos , seu projeto DEVE atender a TODOS os seguintes requisitos:

  • 1. Aplicativo Completo e Funcional Desenvolver um aplicativo mobile real e funcional usando React Native + Expo. O app deve ter um propósito claro e resolver algum problema ou atender alguma necessidade.
  • 2. Múltiplas Telas (Mínimo 4 telas) O aplicativo deve ter pelo menos 4 telas diferentes e funcionais: - Tela inicial/home - Tela de listagem de dados - Tela de detalhes - Tela de configurações ou perfil
  • 3. Navegação entre Telas (React Navigation) Implementar navegação profissional usando React Navigation com: - Stack Navigator (navegação em pilha) - Tab Navigator (navegação por abas) - OPCIONAL mas recomendado, ou qualquer outro tipo de navegação. - Passagem de parâmetros entre telas - Cabeçalhos personalizados
  • 4. Lista de Dados com FlatList Exibir dados em lista usando FlatList com: - Pelo menos 10 itens de dados - Renderização otimizada - Estilização profissional de cada item - Scroll infinito funcionando corretamente

Importante

Projetos que não atenderem a TODOS os requisitos obrigatórios (1 a 4) receberão pontuação proporcional reduzida ou poderão não ser aceitos.

Funcionalidades que seu App Deve Ter

Tela Inicial

  • Apresentação do app
