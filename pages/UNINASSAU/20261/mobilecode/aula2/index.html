🤖 Android Studio - IDE Oficial

✨ Características Principais

  • IntelliJ base: IDE poderosa
  • Gradle build: Sistema moderno
  • Layout Editor: Design visual
  • AVD Manager: Emuladores
  • Profiler: Performance analysis
  • APK Analyzer: Análise de apps

🔧 Componentes Incluídos

  • Android SDK: Ferramentas de desenvolvimento
  • Android NDK: Código nativo (C/C++)
  • Emuladores: Múltiplas versões Android
  • ADB: Android Debug Bridge
  • Lint: Análise de código
  • ProGuard: Ofuscação de código

💾 Requisitos de Sistema

  • RAM: 8GB mínimo (16GB recomendado)
  • Espaço: 10GB+ de armazenamento
  • OS: Windows 10+, macOS 10.14+, Linux Ubuntu 18.04+

🍎 Xcode - Ecossistema Apple

🌟 Recursos Principais

  • Interface Builder: Design visual
  • Swift/Objective-C: Linguagens nativas
  • Simulator: Emulador iOS/watchOS
  • Instruments: Profiling avançado
  • SwiftUI: Framework moderno de UI
  • Core Data: Persistência de dados

📱 Plataformas Suportadas

  • iOS: iPhone applications
  • iPadOS: iPad optimized
  • macOS: Desktop applications
  • watchOS: Apple Watch
  • tvOS: Apple TV
  • visionOS: Apple Vision Pro

✅ Vantagens

  • Integração perfeita com Apple
  • Performance otimizada
  • Ferramentas avançadas incluídas
  • Documentação excelente

❌ Limitações

  • Apenas em macOS
  • Tamanho grande (15GB+)
  • Curva de aprendizado
  • Hardware Apple necessário

💻 Visual Studio Code - Editor Universal

🚀 Por que VS Code para Mobile?

  • Multiplataforma: Windows, macOS, Linux
  • Extensões: React Native, Flutter, Dart
  • Leve: Rápido e eficiente
  • Integração Git: Controle de versão
  • Terminal integrado: Linha de comando
  • IntelliSense: Autocompletar

🔌 Extensões Essenciais

  • React Native Tools: Microsoft
  • Flutter: Dart Code
  • Prettier: Formatação de código
  • ESLint: Linting JavaScript
  • GitLens: Git super powers
  • Bracket Pair: Colorização

📱 Emuladores e Simuladores

🤖 Android Virtual Device (AVD)

  • AVD Manager: Gerenciamento centralizado
  • Google Play Services: APIs do Google
  • Hardware acceleration: HAXM, Hyper-V
  • Diferentes resoluções: Phones, tablets, watches
  • API levels: Android 4.4 até mais recente

🍎 iOS Simulator

  • Xcode incluído: Múltiplos dispositivos
  • iPhone/iPad: Várias gerações
  • Hardware features: Location, Camera, Touch ID
  • iOS versions: Múltiplas versões
  • Network conditions: Simulação de rede

🔧 Alternativas de Emulação

Genymotion

Emulador Android rápido e poderoso. Versão gratuita e paga.

BlueStacks

Emulador Android popular para games e apps. Foco em performance.

Browser DevTools

Simulação mobile em navegadores. Bom para PWAs e web apps.

📲 Teste em Dispositivos Físicos

✅ Vantagens dos Dispositivos Reais

  • Performance real: CPU, RAM, GPU reais
  • Sensores autênticos: GPS, acelerômetro, câmera
  • Conectividade real: WiFi, 4G/5G, Bluetooth
  • Bateria: Impacto no consumo
  • UX autêntica: Touch real, haptic feedback
  • Notificações: Comportamento real

⚙️ Configuração para Debug

🤖 Android:

  • Developer Options: Habilitar modo desenvolvedor
  • USB Debugging: Conectar via ADB
  • OEM Unlocking: Para testes avançados

🍎 iOS:

  • Apple Developer Account: Necessário
  • Device Registration: UDID registration
  • Provisioning Profile: Certificados

📋 Checklist para Testes

  • ✅ Diferentes tamanhos de tela e resoluções
  • ✅ Versões de OS variadas (Android 8+ / iOS 13+)
  • ✅ Diferentes fabricantes (Samsung, Xiaomi, OnePlus)
  • ✅ Condições de rede variadas (WiFi, 4G, offline)

⚛️ React Native & Expo Snack

🚀 React Native

  • Learn Once, Write Anywhere: Filosofia
  • JavaScript/TypeScript: Linguagens conhecidas
  • Native Components: Performance nativa
  • Hot Reload: Desenvolvimento rápido
  • Large Community: Libs e suporte

📦 Instalação

# Instalação via npm
npm install -g react-native-cli

# Criar novo projeto
npx react-native init MeuApp

# Executar no Android
npx react-native run-android
