📘 Introdução ao GDScript

GDScript é a linguagem nativa da Godot Engine, inspirada em Python 🐍.

  • Fácil leitura e escrita
  • Fortemente tipada com inferência
  • Integração direta com os nós da cena

📜 Estrutura Básica de um Script

extends Node2D

func _ready():
    print("O jogo começou!")
  • extends: herança do tipo de nó
  • _ready(): executado ao iniciar a cena
  • _process(delta): chamado a cada frame 🎥

🧮 Variáveis e Tipos

var velocidade = 200
var nome: String = "Player"
const GRAVIDADE = 9.8

Usamos var para variáveis e const para constantes.

🛠️ Funções em GDScript

Funções definem comportamentos dos objetos:

func mover(valor):
    position.x += valor

_ready() e _process(delta) são chamadas pela engine.

🕒 O que é delta?

delta é o tempo entre frames. Permite movimento suave!

func _process(delta):
    position.x += velocidade * delta

Evita variações de movimento em diferentes FPS 🎮

🌳 Manipulação de Nós

  • get_node() para acessar outros nós
  • add_child() para adicionar filhos
  • queue_free() para remover da cena

📡 Sinais (Signals)

Permitem comunicação entre nós sem acoplamento direto.

signal coletado

func _on_colisao():
    emit_signal("coletado")

Conexão é feita na aba Node do editor 🧩

🎮 Movimento e Coleta

func _process(delta):
    var direcao = Vector2.ZERO
    if Input.is_action_pressed("ui_right"):
        direcao.x += 1
    position += direcao.normalized() * velocidade * delta

💰 Coleta de Itens

func _on_Moeda_body_entered(body):
    if body.name == "Player":
        queue_free()
        print("Moeda coletada!")
