Sobre Mim

Professor, Pesquisador e Analista de Dados

Professor, pesquisador e analista de dados com ampla experiência no ensino superior, desenvolvimento de software e análise de dados. Atuo na formação de profissionais de tecnologia na UNINASSAU, ministrando disciplinas em cursos de tecnologia e orientando alunos em projetos de pesquisa e iniciação científica.

Atualmente trabalho como Analista de Dados em uma empresa de seguros, onde estruturo soluções em SQL, desenvolvo relatórios estratégicos e implemento regras de negócios para otimizar a tomada de decisão. Meu foco está na análise e modelagem de dados para garantir eficiência e precisão nas operações.

Destaco-me na estruturação de conteúdos didáticos inovadores, utilizando metodologias ativas e ferramentas modernas.

Google Earth Engine React JavaScript T-SQL Python React Native Node.js MongoDB

Sempre busco tornar o aprendizado prático e aplicável ao mercado, conectando teoria e prática de forma dinâmica.

Coordeno projetos de pesquisa focados em inclusão digital, como o projeto "Programação Acessível: Introdução ao JavaScript para Surdos", e projetos de ciência de dados para identificação de áreas com deficiência de saneamento básico no Recife, utilizando técnicas de machine learning e mineração de dados.
